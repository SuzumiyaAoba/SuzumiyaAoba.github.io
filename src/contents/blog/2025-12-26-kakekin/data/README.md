## データ構造

### JSON フォーマット

各カテゴリファイルは以下の構造を持っています：

```json
{
  "category": "カテゴリ名",
  "sheets": {
    "シート番号": {
      "metadata": {
        "title": "完全なタイトル",
        "toc_title": "目次でのタイトル"
      },
      "headers": [
        "列1のヘッダー",
        "列2のヘッダー",
        ...
      ],
      "series": [
        {
          "year": "1963",
          "values": {
            "列1のヘッダー": 77.8,
            "列2のヘッダー": 22.2,
            ...
          }
        },
        ...
      ]
    }
  }
}
```

### データの読み込み例

#### JavaScript/TypeScript
```javascript
// カテゴリファイルを読み込み
const assetsData = await fetch('./data/assets.json').then(r => r.json());

// シート3（金融資産保有額）のデータを取得
const sheet3 = assetsData.sheets['3'];
console.log(sheet3.metadata.title);

// 年ごとのデータを処理
sheet3.series.forEach(yearData => {
  console.log(`${yearData.year}年: 平均 ${yearData.values['平均']} 万円`);
});

// Chart.js 用のデータを準備
const chartData = {
  labels: sheet3.series.map(d => d.year),
  datasets: [{
    label: '平均保有額',
    data: sheet3.series.map(d => d.values['平均'])
  }]
};
```

#### Python
```python
import json

# カテゴリファイルを読み込み
with open('data/assets.json', 'r', encoding='utf-8') as f:
    assets_data = json.load(f)

# シート3のデータを取得
sheet3 = assets_data['sheets']['3']
print(sheet3['metadata']['title'])

# Pandas DataFrameに変換
import pandas as pd

df = pd.DataFrame([
    {'year': item['year'], **item['values']}
    for item in sheet3['series']
])

# 可視化
import matplotlib.pyplot as plt
plt.plot(df['year'], df['平均'])
plt.xlabel('年')
plt.ylabel('平均保有額（万円）')
plt.show()
```

## 主要な調査項目

### 金融資産関連
- 預貯金口座・証券口座の保有状況
- 金融資産保有額（平均・中央値）
- 種類別金融商品保有額

### 投資商品
- NISA（新NISA・旧NISA）
- 企業型DC・個人型DC（iDeCo）
- 外貨建金融商品

### 貯蓄・運用
- 年間手取り収入からの振り分け割合
- 金融資産構成の前年比較

### 借入金
- 借入金の有無・残高
- 新規借入額
- 借入の目的

### その他
- 決済手段（現金・クレジットカード・電子マネー）
- 生活設計・家計運営
- 世帯属性（年齢・年収・職業など）

## データソース

- **出典**: 金融広報中央委員会「家計の金融行動に関する世論調査」
- **対象**: 二人以上世帯
- **期間**: 1963年〜2025年（項目により異なる）

## 注意事項

- データは元の Excel ファイルから抽出されており、一部の値は `null` （データなし）の場合があります
- "-" で表示されていた箇所は `null` に変換されています
- ヘッダー名は元のExcelファイルの複数行を結合したものです

## ライセンス

元データの著作権は金融広報中央委員会に帰属します。
