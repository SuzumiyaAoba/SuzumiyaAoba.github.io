## IDE を選ぶ・準備する

JDK の準備ができました。
これで、Java のプログラムを動かす「エンジン」は手に入りました。

この節からは、プログラムを書きやすくするための **IDE** を準備します。

第1節で説明したように、IDE は「統合開発環境」、つまりプログラミングに必要な機能を一つにまとめた道具でした。
入力補完や文法チェックなど、初心者にとって心強い手助けをしてくれます。

ここではまず、どんな IDE があるのかを見て、そのあと本書で使う **IntelliJ IDEA**（インテリジェイ・アイディア）をインストールします。

---

## 代表的な IDE

Java の開発でよく使われる IDE には、主に次の3つがあります。

| IDE | 特徴 |
|---|---|
| **IntelliJ IDEA** | Java 開発で最も広く使われている。入力補完が賢く、初心者にもやさしい。本書ではこれを使う |
| **Visual Studio Code（VS Code）** | 軽快で人気の高いエディタ。拡張機能を入れると Java も書ける |
| **Eclipse** | 古くからある定番。長年、多くの現場で使われてきた |

どれを使っても、Java の学習はできます。
そのうえで、本書では **IntelliJ IDEA** を使います。

### 本書が IntelliJ IDEA を選ぶ理由

- **初心者にやさしい** ― 入力補完や間違いの指摘が賢く、書きながら学びやすい
- **無料で使える** ― 学習に必要な機能が、無料で利用できる
- **設定が少なくて済む** ― プロジェクトを作れば、すぐに書き始められる
- **利用者が多い** ― 困ったときに、情報を見つけやすい

> **補足: 「すでに VS Code を使っている」場合**
>
> 普段から VS Code を使っていて、それを使いたい場合は、拡張機能の **「Extension Pack for Java」** を入れると、Java を書けるようになります。
> ただし本書では、説明を IntelliJ IDEA に統一しています。
> 特にこだわりがなければ、IntelliJ IDEA を使うのがスムーズです。

---

## IntelliJ IDEA は無料で使える

IntelliJ IDEA をインストールする前に、料金について整理しておきましょう。
ここは少し変化のあった部分なので、丁寧に説明します。

IntelliJ IDEA には、長らく次の2つの版がありました。

- **Community Edition（コミュニティ版）** … 無料
- **Ultimate Edition（アルティメット版）** … 有料（Web 開発など、高度な機能を含む）

しかし、**2025 年末（バージョン 2025.3）からは、この2つが一つにまとめられました**[^intellij-unified-2025-3]。
現在は、ダウンロードする IntelliJ IDEA は**1種類だけ**です。

新しくなった IntelliJ IDEA の料金は、次のようになっています。

- **基本的な機能（従来の Community 版にあたる部分）は、誰でも無料で使える**
  - 個人の学習でも、仕事でも、無料で利用できます
  - 料金の登録やアクティベーション（有効化の手続き）は不要です
- **より高度な機能（従来の Ultimate 版にあたる部分）は、有料の契約で使える**
  - 30 日間は無料で試せます

つまり、私たちが Java の学習で使う機能は、**すべて無料で使える**ということです。
有料の契約をする必要はありません。

> **補足: 古い情報に注意**
>
> インターネットで検索すると、「IntelliJ IDEA Community Edition をダウンロードする」という、少し前の手順が見つかることがあります。
> 現在は版が一つにまとまっているため、本書では「IntelliJ IDEA を1つダウンロードし、無料の範囲で使う」という、新しい形で説明します。
> もし古いバージョンを使う場合でも、「無料で使える Community 版がある」という考え方は同じです。

## IntelliJ IDEA をインストールする

それでは、IntelliJ IDEA をインストールしましょう。

### 1. ダウンロードする

1. ブラウザで [https://www.jetbrains.com/idea/](https://www.jetbrains.com/idea/) を開きます。
2. ダウンロードのページに進みます。
3. 自分の OS（Windows / macOS）に合った IntelliJ IDEA をダウンロードします。
   - 自分の OS が自動で選ばれていることが多いですが、念のため確認しましょう。
   - Mac の場合は、CPU（Apple Silicon か Intel か）に合ったものを選びます。わからなければ、第2節の「自分の Mac がどちらかわからないとき」を参照してください。

ダウンロードは無料で、メールアドレスの登録なども必要ありません。

### 2. インストールする

ダウンロードしたファイルを開いて、インストールします。

**Windows の場合**

1. ダウンロードした `.exe` ファイルをダブルクリックします。
2. 画面の案内に従って「Next（次へ）」で進みます。
3. インストール先などは、そのままで問題ありません。「Install」を押します。
4. 完了したら、IntelliJ IDEA を起動します。

**macOS の場合**

1. ダウンロードした `.dmg` ファイルを開きます。
2. 表示された **IntelliJ IDEA のアイコン** を、**Applications（アプリケーション）フォルダ** にドラッグ＆ドロップします。
3. 「アプリケーション」フォルダから IntelliJ IDEA を起動します。
4. 初回起動時に「開いてもよいか」と確認されたら、「開く」を選びます。

### 3. 初回起動の設定

初めて起動すると、いくつか設定を聞かれることがあります。

- **テーマ（見た目）の選択** … 明るい見た目（Light）か、暗い見た目（Dark）を選べます。好みでかまいません。あとから変更もできます。
- **利用規約への同意** … 内容を確認して同意すると、先に進めます。

これらは学習に大きく影響しないので、深く考えず進めて大丈夫です。

設定が終わると、**ウェルカム画面**（最初のメニュー画面）が表示されます。
ここから、新しいプロジェクトを作ったり、既存のプロジェクトを開いたりできます。

> **補足: メニューを日本語にしたい・文字を大きくしたい**
>
> IntelliJ IDEA のメニューは、初期状態では英語です。
> 英語のままでも学習に支障はありませんが、日本語にしたい場合は、**日本語化のためのプラグイン**（Japanese Language Pack）を追加できます。
> 「Settings（設定）」→「Plugins（プラグイン）」で `Japanese` を検索し、見つかったものを入れて再起動すると、メニューが日本語になります。
>
> また、画面の文字が小さく感じるときは、「Settings」→「Editor」→「Font」で、コードを表示する文字の大きさを変えられます。
> 自分が読みやすい大きさに調整しておくと、学習が快適になります。

## IntelliJ IDEA は、JDK の場所を知っている必要がある

IDE は「書きやすくする道具」であって、JDK の代わりにはならない ―― 第1節で、そう説明しました。

そのため、IntelliJ IDEA でプログラムを動かすには、IntelliJ に対して **「使う JDK はこれだよ」と教える**必要があります。

ありがたいことに、私たちはすでに第3節で **Temurin の JDK** をインストールしています。
そのため、次の節でプロジェクトを作るときに、その JDK を選ぶだけで済みます。

> **補足: IntelliJ から JDK を入れることもできる**
>
> 実は IntelliJ IDEA には、JDK をその場でダウンロードしてくれる機能もあります。
> プロジェクト作成の画面で、JDK を選ぶ欄から「**Download JDK**（JDK をダウンロード）」を選ぶと、Eclipse Temurin などの一覧から JDK を選んで入れられます。
>
> つまり、JDK のインストールを IntelliJ に任せることもできるのです。
> ただし本書では、JDK と IDE が別物であることをしっかり理解してもらうために、先に JDK を自分で入れる方法を採りました。
> どちらの方法で入れた JDK でも、同じように使えます。

## まとめ

この節では、プログラムを書きやすくする IDE を準備しました。

- Java でよく使われる IDE には、IntelliJ IDEA・VS Code・Eclipse などがある
- 本書では、初心者にやさしい **IntelliJ IDEA** を使う
- IntelliJ IDEA は、現在では版が一つにまとまっており、**学習に必要な機能はすべて無料**で使える（アクティベーション不要）
- 公式サイト（[jetbrains.com/idea](https://www.jetbrains.com/idea/)）から、自分の OS・CPU に合ったものをダウンロードしてインストールする
- IDE は JDK の代わりにはならないので、使う **JDK の場所を IntelliJ に教える**必要がある（次の節で行う）
- IntelliJ には JDK を自分でダウンロードする機能もあるが、本書では先に入れた Temurin を使う

次の節では、いよいよ IntelliJ IDEA で最初のプロジェクトを作り、画面に「Hello, World!」と表示します。
