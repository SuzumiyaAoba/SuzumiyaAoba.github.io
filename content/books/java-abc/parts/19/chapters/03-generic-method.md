## ジェネリックメソッド

ジェネリクスは、クラス全体だけでなく、**メソッド1つだけ**に使うこともできます。
これを**ジェネリックメソッド**（Generic Method）と呼びます。
この節では、その書き方を学びます。

---

## メソッドに型パラメータを付ける

ジェネリックメソッドは、**戻り値の型の前**に、`<T>` という型パラメータを書きます。

例として、「配列の最初の要素を返す」メソッドを、どんな型の配列にも使えるように書いてみましょう。

```java
static <T> T firstOf(T[] array) {
    return array[0];
}
```

`static` の次の **`<T>`** が、「このメソッドは、`T` という型を受け取ります」という宣言です。
そのうしろは、これまでのメソッドと同じ形です。

- `<T>` … 型パラメータの宣言（戻り値の型の前に書く）
- `T`（戻り値の型） … `T` 型の値を返す
- `T[] array`（引数） … `T` 型の配列を受け取る

これで、`firstOf` は「`T` 型の配列を受け取り、その最初の要素（`T` 型）を返す」メソッドになりました。
`T` が何になるかは、呼び出すときに、渡す配列によって自動的に決まります。

---

## 使ってみる

`firstOf` を、いろいろな型の配列で使ってみましょう。

```java
public class Main {
    static <T> T firstOf(T[] array) {
        return array[0];
    }

    public static void main(String[] args) {
        String[] names = { "佐藤", "鈴木", "高橋" };
        String firstName = firstOf(names);      // T = String
        IO.println(firstName);

        Integer[] numbers = { 10, 20, 30 };
        Integer firstNumber = firstOf(numbers); // T = Integer
        IO.println(firstNumber);
    }
}
```

```text
$ java Main.java
佐藤
10
```

`firstOf(names)` では、`String[]` を渡したので `T` は `String` になり、`String` が返ります。
`firstOf(numbers)` では、`Integer[]` を渡したので `T` は `Integer` になります。

注目してほしいのは、呼び出すとき、`<String>` のような型指定を**書いていない**ことです。
ジェネリックメソッドでは、渡した引数（`names` は `String[]`）から、`T` が `String` だと、Java が**自動的に推論**してくれます。
そのため、ふつうのメソッドと同じ感覚で、`firstOf(names)` と呼ぶだけで済みます。

---

## ジェネリッククラスとの違い

ジェネリッククラスとジェネリックメソッドは、似ていますが、型パラメータの「適用範囲」が違います。

| | 型パラメータの場所 | 範囲 |
|---|---|---|
| ジェネリッククラス | クラス名のうしろ（`class Box<T>`） | そのクラス全体で `T` を使える |
| ジェネリックメソッド | メソッドの戻り値の型の前（`<T> T firstOf(...)`） | そのメソッドの中だけで `T` を使える |

- **クラス全体**で同じ型を扱いたい（箱に入れる型など） → ジェネリッククラス
- **そのメソッドだけ**で型を共通化したい（引数と戻り値の型をそろえるなど） → ジェネリックメソッド

`firstOf` のように、「いろいろな型の配列に対して、同じ処理をしたい」場合は、メソッドだけをジェネリックにするのが、ちょうどよいのです。

---

## まとめ

- **ジェネリックメソッド**は、メソッド1つだけに型パラメータを付ける
- 書き方は、**戻り値の型の前**に `<T>` を書く（`static <T> T firstOf(T[] array)`）
- 呼び出すとき、型は**引数から自動推論**される（`<String>` を書かなくてよい）
- クラス全体で型を扱うなら**ジェネリッククラス**、メソッドだけなら**ジェネリックメソッド**
