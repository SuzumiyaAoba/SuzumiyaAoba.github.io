## テキストブロックとは ― 複数行の文字列

**テキストブロック**（Text Block）は、`"""`（ダブルクォート3つ）で囲んで、**複数行の文字列を、見たままに書ける**しくみです。
この節では、その基本を学びます。

---

## これまでの複数行文字列の、つらさ

第10章で学んだ、ふつうの文字列（`"..."`）で、複数行を表すには、改行を表す **`\n`** を、自分で書く必要がありました。

```java
String html = "<ul>\n  <li>りんご</li>\n  <li>みかん</li>\n</ul>";
IO.println(html);
```

```text
<ul>
  <li>りんご</li>
  <li>みかん</li>
</ul>
```

表示結果は正しいのですが、**コードがとても読みにくい**ですね。
`\n` があちこちに入り、`"` で囲むためのエスケープ（`\"`）も必要で、「最終的にどんな文字列になるのか」が、ぱっと見では分かりません。

---

## テキストブロックなら、見たまま

**テキストブロック**を使うと、同じ文字列を、**見たままに**書けます。
`"""`（ダブルクォート3つ）で囲み、その中に、複数行をそのまま書きます。

```java
String html = """
    <ul>
      <li>りんご</li>
      <li>みかん</li>
    </ul>""";
IO.println(html);
```

```text
<ul>
  <li>りんご</li>
  <li>みかん</li>
</ul>
```

`\n` も `\"` も、一切書いていません。
コードに書いた**改行が、そのまま改行**になり、`"` も**そのまま**書けます。
「コードの見た目」と「実際の文字列」が一致するので、HTML や JSON、長いメッセージなどを、とても書きやすくなります。

---

## 書き方のルール

テキストブロックには、いくつかの決まりがあります。

### 開始の `"""` の後は、改行する

開始の `"""` の**直後**には、文字を書けません。必ず改行します。

```java
String s = """
    内容をここから書く""";   // ◯ """ の後で改行している
```

```java
String s = """内容""";   // ✕ """ の直後に文字 → エラー
```

「`"""` で開いたら、いったん改行して、次の行から内容を書く」と覚えておきましょう。

### 終了の `"""` の位置で、見え方が変わる

終了の `"""` は、内容の最後に続けて書くことも、改行して次の行に書くこともできます。これで「**末尾に改行を付けるか**」が変わります。

```java
String a = """
    こんにちは""";       // 末尾に改行なし → "こんにちは"

String b = """
    こんにちは
    """;                 // 末尾に改行あり → "こんにちは\n"
```

`a` は「こんにちは」だけ、`b` は最後に改行が付きます。
細かいですが、「終了の `"""` を、内容の直後に置くか、改行して置くか」で、末尾の改行が変わる、と知っておきましょう。

---

## 使いどころ

テキストブロックは、「**複数行の、決まった形のテキスト**」を書くのに最適です。

- HTML や XML
- JSON
- SQL（データベースへの命令文。第37章）
- 長いメッセージ、定型文

```java
String sql = """
    SELECT name, age
    FROM users
    WHERE age >= 20
    ORDER BY age""";
```

このように、本来の形（改行やインデント）を保ったまま書けるので、読みやすく、間違いも減ります。

> **補足: テキストブロックは Java 15 から**
>
> テキストブロックは **Java 15** で正式機能になりました。
> それ以前は、複数行を `\n` でつなぐしかありませんでした。いまでは、複数行の文字列を書くなら、テキストブロックが定番です。

---

## まとめ

- **テキストブロック**は、**`"""`** で囲んで、**複数行の文字列を見たまま書ける**しくみ
- `\n`（改行）や `\"`（クォートのエスケープ）を書かなくてよい
- 開始の `"""` の**直後は改行**する（直後に文字は書けない）
- 終了の `"""` の位置で、**末尾の改行の有無**が変わる
- HTML・JSON・SQL・長いメッセージなどに最適
