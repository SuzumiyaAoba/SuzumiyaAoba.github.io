## はじめに ― この章で学ぶこと

プログラムでは、文字列を扱う場面が、本当にたくさんあります。
メッセージを組み立てる、複数行の文章を書く、値を決まった形式で表示する ―― 第10章で文字列の基本を学びましたが、この章では、それを**もっと快適に書く**ための機能を学びます。

この第28章のテーマは、大きく2つです。

1. **テキストブロック**（Text Block） … **複数行の文字列**を、見たままに書ける
2. **文字列フォーマット** … 値を埋め込んで、**決まった形式に整形**する

たとえば、複数行の文章を、これまでは `\n`（改行）だらけで書く必要がありました。

```java
// これまで（読みにくい）
String message = "拝啓\n時下ますますご清栄のこととお慶び申し上げます。\nさて、…";
```

テキストブロックを使うと、**見たままに**書けます。

```java
// テキストブロック（見たまま）
String message = """
    拝啓
    時下ますますご清栄のこととお慶び申し上げます。
    さて、…""";
```

`\n` を1つも書かずに、改行がそのまま反映されます。とても読みやすくなりました。

## この章で学ぶこと

第28章は、次の6つの節で構成されています。

| 節 | タイトル | 内容 |
|---|---|---|
| 1 | テキストブロックとは | 複数行の文字列を見たまま書く |
| 2 | テキストブロックの細かいルール | インデント・改行の扱い |
| 3 | 文字列の整形 | `formatted`・`String.format` で値を埋め込む |
| 4 | 文字列を組み立てる | `String.join`・`StringBuilder` |
| 5 | よくあるつまずき | インデント・効率 など |

前半（1〜2節）で**テキストブロック**、後半（3〜4節）で**文字列の整形・組み立て**を学びます。

## この章を読み終えると

第28章を読み終えるころには、次のことができるようになっています。

- テキストブロック（`"""`）で、複数行の文字列を見たまま書ける
- テキストブロックのインデントや改行のルールがわかる
- `formatted`・`String.format` で、値を埋め込んで整形できる
- `String.join`・`StringBuilder` で、文字列を効率よく組み立てられる

<Column title="「文字列テンプレート」はなくなった">

この章は、当初「文字列テンプレート」も扱う予定でした。
文字列テンプレート（String Templates）は、`"こんにちは、\{name}さん"` のように、文字列の中に直接、値を埋め込む機能として、**プレビュー機能**（試験的な先行公開）で提供されていました。
ですが、設計を見直すため、**Java 23 でいったん取り下げられ**、本書が対象とする **Java 25 には含まれていません**。
そのため本書では、文字列に値を埋め込むには、`formatted`（第3節）を使います。
（新しい機能でも、こうして取り下げ・再設計されることがあります。プレビュー機能は「将来確定するとは限らない」と知っておきましょう。）

</Column>

それでは、最初の節「テキストブロックとは」から始めましょう。
