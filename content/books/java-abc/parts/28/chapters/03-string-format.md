## 文字列の整形 ― formatted・String.format

「`佐藤さんは25歳です`」のように、**文字列の中に値を埋め込む**ことは、よくあります。
これまでは `+` で連結してきましたが、埋め込む値が増えると、読みにくくなります。
この節では、**書式（フォーマット）**を使って、値をきれいに埋め込む方法を学びます。

---

## + 連結の、読みにくさ

第10章で学んだ `+` で、値を埋め込んでみましょう。

```java
String name = "佐藤";
int age = 25;
String s = name + "さんは" + age + "歳です";
IO.println(s);
```

```text line-numbers=false
佐藤さんは25歳です
```

これでも書けますが、埋め込む値が増えると、`"` と `+` が入り乱れて、「**最終的にどんな文章になるのか**」が分かりにくくなります。

---

## formatted ― 穴あきテンプレートに、値を入れる

**`formatted(...)`** を使うと、「**穴のあいたテンプレート**」を用意して、そこに値を流し込めます。
穴の部分は、**`%s`**（文字列用）や **`%d`**（整数用）といった**書式指定子**で表します。

```java
String name = "佐藤";
int age = 25;
String s = "%sさんは%d歳です".formatted(name, age);
IO.println(s);
```

```text line-numbers=false
佐藤さんは25歳です
```

`"%sさんは%d歳です"` というテンプレートの、`%s` に `name`（佐藤）、`%d` に `age`（25）が、順番に入りました。
`+` と `"` を交互に書くより、「**文章の形**」が、ぱっと見て分かりますね。
穴（`%s`・`%d`）の位置に、引数が**左から順に**入る、と覚えてください。

---

## String.format ― 同じことの、別の書き方

`formatted` と、まったく同じことが、**`String.format(...)`** でもできます。

```java
String s = String.format("%sさんは%d歳です", "佐藤", 25);
IO.println(s);
```

```text line-numbers=false
佐藤さんは25歳です
```

`String.format(テンプレート, 値, 値, ...)` の形です。
`formatted` は「テンプレート文字列に対して呼ぶ」、`String.format` は「`String` クラスの機能として呼ぶ」だけの違いで、結果は同じです。
どちらを使ってもかまいません（`formatted` のほうが新しく、すっきり書けます）。

---

## よく使う書式指定子

`%` で始まる**書式指定子**には、いろいろな種類があります。よく使うものを挙げます。

| 指定子 | 意味 | 例 |
|---|---|---|
| `%s` | 文字列 | `"%s".formatted("佐藤")` → `佐藤` |
| `%d` | 整数 | `"%d".formatted(42)` → `42` |
| `%f` | 小数 | `"%f".formatted(3.14)` → `3.140000` |
| `%.2f` | 小数（桁数指定） | `"%.2f".formatted(3.14159)` → `3.14` |
| `%,d` | 整数（3桁区切り） | `"%,d".formatted(1000000)` → `1,000,000` |
| `%%` | `%` 自身 | `"%d%%".formatted(50)` → `50%` |

特に便利なのが、**桁数の指定**です。

```java
IO.println("%.2f".formatted(3.14159));   // 小数点以下2桁に
IO.println("%,d".formatted(1000000));    // 3桁ごとにカンマ
```

```text line-numbers=false
3.14
1,000,000
```

- `%.2f` … 小数点以下を**2桁**にそろえる（金額や割合の表示に便利）
- `%,d` … 大きな数を、**3桁ごとにカンマ**で区切る（読みやすい金額表示に）

「`%f` のままだと `3.140000` と長いので、`%.2f` で2桁にする」――
こうした細かい整形が、書式指定子でかんたんにできます。

---

## 幅をそろえる

数字や文字の**幅**をそろえて、表のように整えることもできます。

```java
IO.println("%5d".formatted(42));    // 5文字幅（右寄せ）
IO.println("%-5d|".formatted(42));  // 5文字幅（左寄せ）
```

```text line-numbers=false
   42
42   |
```

- `%5d` … 全体が**5文字**になるよう、左に空白を入れる（右寄せ）
- `%-5d` … 同じく5文字幅で、**左寄せ**（`-` を付ける）

数値を縦にそろえて表示したいとき（一覧表など）に役立ちます。

---

## まとめ

- 値の埋め込みは、`+` 連結より、**書式（フォーマット）**を使うと読みやすい
- **`"テンプレート".formatted(値, ...)`** … 穴（`%s`・`%d`）に値を順に流し込む
- **`String.format(テンプレート, 値, ...)`** … 同じことの別の書き方（結果は同じ）
- 書式指定子：`%s`（文字列）・`%d`（整数）・`%f`（小数）・`%.2f`（桁指定）・`%,d`（3桁区切り）
- `%5d` で幅をそろえる（`-` で左寄せ）
