## はじめに ― この章で学ぶこと

**Observer**（オブザーバー、観察者）パターンは、**「あるオブジェクトの変化を、別のオブジェクトたちに自動的に通知する」**振る舞いパターンです。

「YouTube のチャンネル登録」「ニュースの購読」「イベントリスナー」「リアクティブストリームの subscribe」 ―― 名前は違えど、すべて Observer パターンの応用です。

---

## この章を読み終えると

- Observer の**意図**を、自分の言葉で説明できる
- 古典的な Observer の実装が書ける
- Java の **`Flow` API**（Reactive Streams 準拠）の位置づけがわかる
