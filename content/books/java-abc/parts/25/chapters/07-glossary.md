## 用語集 ― この章で学んだ言葉

第25章では、日付・時刻を扱う「java.time」を学びました。
節ごとに、言葉を整理しておきます。

---

## 基本クラス（第1〜2節）

| 用語 | 英語・読み | 意味 |
|---|---|---|
| java.time | ― | Java 8 で導入された、モダンな日付・時刻 API |
| `LocalDate` | ― | 日付だけ（`LocalDate.of(2026, 6, 9)`、月は1から） |
| `LocalTime` | ― | 時刻だけ（`LocalTime.of(14, 30)`） |
| `LocalDateTime` | ― | 日付＋時刻（区切りは `T`） |
| `now()` | ― | 今日・現在を取得する |

`java.time` は不変。月は1から（1月＝1）。

---

## 計算（第3節）

| 用語 | 意味 |
|---|---|
| `plusDays` / `minusMonths` など | 日付の足し算・引き算（元は変えず新しい日付を返す） |
| `isBefore` / `isAfter` | 日付の前後を比較する |
| `Period` | 日付の差（年・月・日）。`Period.between(...)` |
| `Duration` | 時刻の差（時・分・秒）。`Duration.between(...)` |
| `ChronoUnit.DAYS.between` | 総日数を求める |

---

## 整形・解析（第4節）

| 用語 | 英語・読み | 意味 |
|---|---|---|
| `DateTimeFormatter` | ― | 日付の表示・読み取りの形式を表す |
| 整形（format） | Format | 日付 → 好きな形式の文字列 |
| 解析（parse） | Parse | 文字列 → 日付 |

形式の記号：`yyyy`（年）・`MM`（月・大文字）・`dd`（日）・`HH`（時）・`mm`（分・小文字）。

---

## タイムゾーン（第5節）

| 用語 | 英語・読み | 意味 |
|---|---|---|
| タイムゾーン | Time Zone | 場所による時差 |
| `ZonedDateTime` | ― | タイムゾーン付きの日時（`ZoneId.of(...)`） |
| `Instant` | ― | 世界共通の「ある瞬間」（UTC 基準、末尾 `Z`） |

---

## おわりに

この章では、日付・時刻を安全に扱う `java.time` を学びました。

- **`LocalDate`・`LocalTime`・`LocalDateTime`** … 目的別の基本クラス（月は1から、すべて不変）
- **`plus`・`minus`・`Period`・`Duration`** … 計算と差
- **`DateTimeFormatter`** … 表示と読み取り
- **`ZonedDateTime`・`Instant`** … 世界の時刻

日付・時刻という、身近だけれど落とし穴の多いテーマを、安全に扱えるようになりました。
