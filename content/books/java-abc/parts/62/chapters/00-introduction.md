## はじめに ― この章で学ぶこと

**Flyweight**（フライウェイト、軽量級）パターンは、**「同じ状態のオブジェクトを共有することで、数を減らしてメモリを節約する」**ためのパターンです。

「**たくさんの似たオブジェクト**」 ―― たとえば、文書のすべての文字、ゲーム内の大量のパーティクル、地図の道路の点 ―― を効率よく表現するときに使われます。

Java の `Integer.valueOf` や `String#intern`、`Boolean.TRUE` などにも、この発想が組み込まれています。

---

## この章を読み終えると

- Flyweight の**意図**を、自分の言葉で説明できる
- **内部状態と外部状態**を分けて考えられる
- Java 標準にひそむ Flyweight を見抜ける
