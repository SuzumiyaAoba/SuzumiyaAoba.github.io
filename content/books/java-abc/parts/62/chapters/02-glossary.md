## 用語集 ― この章で学んだ言葉

| 用語 | 英語 | 意味 |
|---|---|---|
| Flyweight | Flyweight | 同じ状態のオブジェクトを共有してメモリを節約する構造パターン |
| 内部状態 | Intrinsic State | 共有してよい不変属性 |
| 外部状態 | Extrinsic State | 共有できず、利用側が持つ状況依存の属性 |
| プール / キャッシュ | Pool / Cache | 共有インスタンスを保持する仕組み |
| インターン化 | Interning | 同じ値の文字列に同じインスタンスを使い回すこと |

---

## この章で学んだこと

- **Flyweight** は、状態の共有でオブジェクト数を抑える構造パターン
- 内部状態と外部状態の**分離**がパターン適用のカギ
- Java 標準（`Integer.valueOf`、`String#intern`）に組み込まれた身近な例
- 早すぎる最適化に注意

これで構造パターン（第56〜62章）の解説は完了です。
