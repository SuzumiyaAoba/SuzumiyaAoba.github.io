## 用語集 ― この章で学んだ言葉

この第6章で登場した言葉を、まとめておきます。

---

## 条件分岐の基本（第1〜4節）

| 用語 | 英語・読み | 意味 |
|---|---|---|
| 条件分岐 | じょうけんぶんき | 条件が成り立つかどうかで、実行する処理を変えるしくみ |
| if 文 | イフ | 「もし条件が成り立つなら、この処理をする」を表す |
| else | エルス | 「そうでなければ」。条件が `false` のときの処理 |
| else if | ― | 3つ以上に枝分かれさせるための書き方 |
| ブロック | Block | `{ }` で囲んだ処理のまとまり |
| ネスト | Nest | if の中に if を書く、入れ子の構造 |
| 複合条件 | ― | `&&` や `\|\|` で複数の条件を1つにまとめたもの |

---

## switch（第5節）

| 用語 | 英語・読み | 意味 |
|---|---|---|
| switch 式 | スイッチ | `->` を使い、値によって結果を選ぶ。全体で1つの値になる（Java 14 以降） |
| switch 文 | スイッチ | `:` と `break` を使う、従来の書き方 |
| case | ケース | 照合する値ごとの枝 |
| default | デフォルト | どの `case` にも当てはまらないときの枝 |
| break | ブレーク | 従来の switch 文で、その枝の終わりを示す。書き忘れると次へ流れる |
| yield | イールド | switch 式の `{ }` ブロック内で、返す値を指定する |
| フォールスルー | Fall-through | switch 文で `break` を忘れ、次の `case` へ処理が流れ込むこと |
| 網羅 | もうら | switch 式が、起こりうるすべての場合をカバーすること（`default` などで保証） |

---

## おわりに

この第6章では、プログラムの流れを枝分かれさせる**条件分岐**を学びました。

- **if 文**で「もし〜なら」を表し、`else`・`else if` で枝を増やす
- `{ }`（ブロック）でまとまりを作り、`&&`・`||` で複数条件をまとめる
- **switch 式**で、1つの値による多くの分岐を、すっきり書く

これで、プログラムは「状況に応じて動きを変える」ことができるようになりました。
次の第7章「**繰り返し（for / while / do-while）**」では、もうひとつのプログラムの基本 ―― **同じ処理を何度も繰り返す**方法を学びます。条件分岐と繰り返しがそろえば、表現できる処理が一気に広がります。
