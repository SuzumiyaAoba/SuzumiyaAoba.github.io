## if 文 ― もし〜なら

条件分岐の基本となるのが、**if 文**（イフぶん）です。
`if` は英語で「もし」という意味で、「もし条件が成り立つなら、この処理をする」を表します。

---

## if の基本の形

if 文は、次のように書きます。

```java
if (条件) {
    // 条件が true のときに実行する処理
}
```

`if` のうしろのかっこ `( )` に**条件**（`boolean` になる式）を書き、`{ }` の中に、条件が成り立ったときの処理を書きます。

実際の例を見てみましょう。点数が 60 以上なら「合格」と表示します。

```java
int score = 80;
if (score >= 60) {
    IO.println("合格");
}
```

`score` は 80 で、`score >= 60` は `true` なので、`{ }` の中が実行されます。

```text
合格
```

もし `score` が 50 だったら、`score >= 60` は `false` なので、`{ }` の中は実行されず、何も表示されません。

---

## else ― そうでなければ

「条件が成り立たないときにも、別の処理をしたい」――そんなときは、**`else`**（エルス、そうでなければ）を続けます。

```java
int score = 50;
if (score >= 60) {
    IO.println("合格");
} else {
    IO.println("不合格");
}
```

- `score >= 60` が `true` なら、`if` のブロックが実行される
- `false` なら、`else` のブロックが実行される

`score` は 50 なので `false`、`else` のほうが実行されます。

```text
不合格
```

`if` と `else` で、**かならずどちらか一方だけ**が実行されます。

この「条件で道が分かれる」ようすを図にすると、次のようになります。

![if と else の分かれ道。条件 score >= 60 が true なら「合格」へ、false なら「不合格」へ進む。条件によって、必ずどちらか一方の道だけを通る。](./images/if-else-flow.svg)

条件が `true` のときは「合格」の道へ、`false` のときは「不合格」の道へ進みます。両方の道を同時に通ることはありません。

> **補足: 条件には「`boolean` になる式」を書く**
>
> `if ( )` のかっこの中に書けるのは、結果が `true` / `false`（`boolean`）になる式だけです。
> `score >= 60` のような比較はもちろん、`boolean` 型の変数（`if (isMember)` など）もそのまま書けます。
> 一方、`if (score)` のように数値をそのまま条件にすることは、Java ではできません（一部の言語とは違う点です）。「条件＝ `boolean` になる式」と覚えておきましょう。

## もう一つの例 ― 偶数か奇数か

if-else をもう一つ見てみましょう。数が偶数か奇数かを判定します。
第5章で学んだ剰余演算子 `%` を使い、「2 で割った余りが 0 かどうか」で見分けます。

```java
int num = 7;
if (num % 2 == 0) {
    IO.println(num + " は偶数です");
} else {
    IO.println(num + " は奇数です");
}
```

```text
7 は奇数です
```

`7 % 2` は `1` なので、`num % 2 == 0` は `false`。`else` の側が実行され、「奇数です」と表示されました。
このように、if 文の条件には、第5章で学んだ比較演算子や算術演算子を組み合わせた、さまざまな式を書けます。

---

## 構造をまとめると

```text
if (条件) {
    条件が true のときの処理
} else {
    条件が false のときの処理
}
```

「もし条件が成り立つなら上を、そうでなければ下を実行する」と読めます。
`else` は省略できます。省略すると「条件が成り立つときだけ処理し、そうでなければ何もしない」という意味になります。

---

## まとめ

- **if 文**は「もし条件が成り立つなら、この処理をする」を表す
- `if (条件) { ... }` の `{ }` は、条件が `true` のときだけ実行される
- **`else`** を続けると、条件が `false` のときの処理を書ける
- `if` と `else` では、かならずどちらか一方だけが実行される
