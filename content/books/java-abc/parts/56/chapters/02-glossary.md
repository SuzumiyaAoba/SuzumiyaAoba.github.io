## 用語集 ― この章で学んだ言葉

| 用語 | 英語 | 意味 |
|---|---|---|
| Adapter | Adapter | 形の合わないクラスを、目的のインターフェースに合わせる構造パターン |
| クラス Adapter | Class Adapter | 継承を使って既存クラスを取り込む形の Adapter |
| オブジェクト Adapter | Object Adapter | 委譲を使って既存オブジェクトをラップする形の Adapter |
| 委譲 | Delegation | 別のオブジェクトに処理を任せる設計手法 |
| Adaptee | Adaptee | Adapter が中で利用する、変換対象の既存クラス |

---

## この章で学んだこと

- **Adapter** は、形の異なるクラスを橋渡しする構造パターン
- 実務では委譲ベースの**オブジェクト Adapter** が基本
- 1 メソッドだけの場合はラムダで十分
