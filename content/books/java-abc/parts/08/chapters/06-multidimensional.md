## 多次元配列

ここまでの配列は、値が**一列に**並んでいました。
しかし、世の中には「縦 × 横」の表のように、**格子状**に並んだデータもあります。
たとえば、表計算ソフトのマス目、座席表、ゲームの盤面などです。

こうしたデータは、**配列の中に配列を入れる**ことで表せます。これを**多次元配列**（たじげんはいれつ）と呼びます。
もっともよく使うのは、2段重ねの**2次元配列**です。

---

## 2次元配列を作る

2次元配列の型は、`[]` を2つ重ねて `int[][]` と書きます。
値を直接並べるときは、`{ }` の中に、さらに `{ }` を並べます。

```java
int[][] grid = {
    {1, 2, 3},
    {4, 5, 6}
};
```

これは「3個の配列が、2つ並んでいる」という構造です。
表でいうと、**2行3列**のイメージです。

```text
        列0  列1  列2
 行0 [   1    2    3  ]
 行1 [   4    5    6  ]
```

jshell で確認すると、配列の中に配列が入っているのがわかります。

```text
jshell> int[][] grid = {{1, 2, 3}, {4, 5, 6}};
grid ==> int[2][] { int[3] { 1, 2, 3 }, int[3] { 4, 5, 6 } }
```

---

## 要素にアクセスする

2次元配列の要素は、`grid[行][列]` と、インデックスを2つ書いて取り出します。
どちらのインデックスも、もちろん **`0` から始まります**。

```java
int[][] grid = {{1, 2, 3}, {4, 5, 6}};
```

```text
jshell> grid[0][2]
$2 ==> 3

jshell> grid[1][0]
$3 ==> 4
```

- `grid[0][2]` … 行0・列2 の要素 → `3`
- `grid[1][0]` … 行1・列0 の要素 → `4`

「`grid[行][列]`」の順番です。最初の `[ ]` が**外側の配列（行）**、次の `[ ]` が**その中の配列（列）**を指します。

---

## 行数と列数を知る

2次元配列の `length` は、少し注意が必要です。

```text
jshell> grid.length
$4 ==> 2

jshell> grid[0].length
$5 ==> 3
```

- `grid.length` … 外側の配列の要素数 ＝ **行数**（ここでは 2）
- `grid[0].length` … 0行目（内側の配列）の要素数 ＝ **列数**（ここでは 3）

「`grid.length` が行数、`grid[行].length` が、その行の列数」と覚えましょう。

---

## 二重ループで全要素をたどる

2次元配列は、第7章で学んだ**多重ループ**と相性が抜群です。
外側のループで行を、内側のループで列をたどります。

```java
int[][] grid = {{1, 2, 3}, {4, 5, 6}};
for (int i = 0; i < grid.length; i++) {
    for (int j = 0; j < grid[i].length; j++) {
        IO.print(grid[i][j] + " ");
    }
    IO.println("");
}
```

```text
1 2 3 
4 5 6 
```

外側の `i`（行）が1つ進むごとに、内側の `j`（列）が、その行の最後まで回ります。
`grid[i][j]` で、i 行 j 列の要素を取り出しています。
内側のループが終わったら `IO.println("")` で改行し、次の行へ移ります。

> **補足: なぜ `grid[i].length` を使うのか**
>
> 内側の条件に `grid[0].length` ではなく `grid[i].length` を使っているのは、行ごとに列数が違う配列（ギザギザの配列）にも対応するためです。
> 多くの場合は全行が同じ列数ですが、`grid[i].length` と書いておけば、どんな形でも安全にたどれます。

---

## 入れ物だけ先に作る

2次元配列も、`new` で「空の入れ物」を先に作れます。
`new int[2][3]` は、「2行3列ぶんの入れ物」を用意し、すべての要素を初期値（`int` なら `0`）で埋めます。

```text
jshell> int[][] table = new int[2][3];
table ==> int[2][] { int[3] { 0, 0, 0 }, int[3] { 0, 0, 0 } }
```

あとから `table[0][1] = 5;` のように、インデックスを2つ指定して値を入れていきます。

---

## まとめ

- **多次元配列**は、配列の中に配列を入れたもの。よく使うのは2段重ねの**2次元配列**（`int[][]`）
- 要素は `grid[行][列]` と、インデックスを2つ指定して取り出す（どちらも 0 始まり）
- `grid.length` が行数、`grid[行].length` がその行の列数
- **二重ループ**（外側で行、内側で列）で全要素をたどるのが定番
- `new int[2][3]` で、空の2次元配列を作れる

次の節では、配列の中身を表示したり並べ替えたりするのに便利な、**`Arrays` クラス**を学びます。
