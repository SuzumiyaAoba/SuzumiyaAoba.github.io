## 引数 ― メソッドに値を渡す

前節の `greet` は、いつも「こんにちは」としか言えませんでした。
でも、「佐藤さん、こんにちは」「鈴木さん、こんにちは」のように、**渡した値によって動きを変えたい**こともあります。

そのために、メソッドへ値を渡すしくみが **引数**（ひきすう、Argument）です。

---

## 引数を受け取るメソッド

名前を受け取って、あいさつするメソッドを作ってみましょう。

```java
void greet(String name) {
    IO.println(name + "さん、こんにちは");
}
```

`()` の中に書いた **`String name`** が、引数です。
「`String` 型の値を1つ受け取り、それを `name` という名前で使う」という意味です。

呼び出すときは、`()` の中に渡したい値を書きます。

```java
greet("佐藤");
greet("鈴木");
```

```text
佐藤さん、こんにちは
鈴木さん、こんにちは
```

`greet("佐藤")` と呼ぶと、`"佐藤"` が `name` に入り、`name + "さん、こんにちは"` が「佐藤さん、こんにちは」になります。
渡す値を変えるだけで、メソッドの動きが変わりました。

---

## 仮引数と実引数

引数には、立場の違う2つの言葉があります。

- **仮引数**（かりひきすう、Parameter）… 定義のほうに書く、受け取り用の変数（`String name` の `name`）
- **実引数**（じつひきすう、Argument）… 呼び出すときに渡す、実際の値（`greet("佐藤")` の `"佐藤"`）

「仮引数 `name` という入れ物に、実引数 `"佐藤"` が入る」とイメージするとわかりやすいでしょう。
仮引数は、メソッドの中で、ふつうの変数として使えます。

---

## 複数の引数を渡す

引数は、カンマ `,` で区切って、いくつでも書けます。
それぞれに**型と名前**を書きます。

```java
void introduce(String name, int age) {
    IO.println(name + "さん（" + age + "歳）");
}
```

呼び出すときは、定義した**順番どおり**に値を渡します。

```java
introduce("佐藤", 25);
```

```text
佐藤さん（25歳）
```

`"佐藤"` が `name` に、`25` が `age` に、書いた順に入ります。
順番や型を間違えると、正しく動かない（またはエラーになる）ので注意しましょう。
たとえば `introduce(25, "佐藤")` のように型の合わない順で渡すと、エラーになります。

---

## まとめ

- **引数**は、メソッドに値を渡すしくみ。`()` の中に「型 名前」で書く
- 定義側の受け取り用変数を**仮引数**、呼び出し側で渡す値を**実引数**と呼ぶ
- 引数はカンマ区切りで複数書ける。渡すときは**定義した順番・型**に合わせる

ここまでのメソッドは、表示するだけで、結果を返しませんでした。
