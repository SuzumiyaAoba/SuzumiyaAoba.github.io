## 用語集 ― この章で学んだ言葉

この第9章で登場した言葉を、まとめておきます。

---

## メソッドの基本（第1〜4節）

| 用語 | 英語・読み | 意味 |
|---|---|---|
| メソッド | Method | 一連の処理に名前をつけてまとめた「部品」 |
| 定義 / 呼び出し | Define / Call | メソッドを作ること／作ったメソッドを実行すること |
| 引数 | Argument（ひきすう） | メソッドに渡す値 |
| 仮引数 | Parameter（かりひきすう） | 定義側の受け取り用の変数（`String name` の `name`） |
| 実引数 | Argument（じつひきすう） | 呼び出し側で渡す実際の値（`greet("佐藤")` の `"佐藤"`） |
| 戻り値 | Return Value（もどりち） | メソッドが呼び出し元へ返す値 |
| return | リターン | 値を返し、メソッドを終える文（`void` では値なしの終了にも使う） |
| void | ボイド | 「戻り値がない」ことを表す型 |

---

## 組み合わせと応用（第5〜7節）

| 用語 | 英語・読み | 意味 |
|---|---|---|
| スコープ | Scope | 変数が使える範囲 |
| ローカル変数 | Local Variable | メソッドの中だけで使える変数。メソッドが終わると消える |
| オーバーロード | Overload | 引数の組み合わせが違えば、同じ名前のメソッドを複数定義できるしくみ |
| シグネチャ | Signature | メソッドを見分ける「名前＋引数の組み合わせ」 |
| 可変長引数 | Varargs | `int...` のように、個数の決まらない引数を受け取る書き方 |
| 値渡し | Pass by Value | 値のコピーを渡すしくみ。Java の引数はすべてこれ |

---

## おわりに

この第9章では、処理を部品にする**メソッド**を学びました。

- メソッドは「**引数（入力）→ 処理 → 戻り値（出力）**」の部品。一度作れば何度でも呼べる
- 大きな処理を小さなメソッドに分けると、各メソッドが1つの役割に集中し、読みやすくなる
- メソッドどうしは**引数と戻り値**でやりとりし、ローカル変数は**スコープ**の中だけで生きる
- 引数は**値渡し**。基本型はコピー、配列などは中身を共有する

これで第1部「プログラミングと Java の基礎」も、残すは次の1章となりました。
次の第10章「**文字列の操作**」では、これまで何気なく使ってきた `String` を、メソッドを通してくわしく扱います。
`length()` や、文字の取り出し・検索・置換など、`String` が持つさまざまなメソッドを学べば、文字を自在にあやつれるようになります。

そして第11章からは、いよいよ第2部「**オブジェクト指向**」へ。
この章で学んだメソッドは、第11章で学ぶ**クラス**の中心的な部品になります。ここでの理解が、これからの土台になります。
