## 戻り値 ― メソッドから値を返す

これまでのメソッドは、画面に表示するだけでした。
しかし、「2つの数を足した**結果**がほしい」「点数から**評価**を求めたい」のように、メソッドに**計算結果を返してほしい**ことがあります。

メソッドが呼び出し元へ返す値を、**戻り値**（もどりち、Return Value）と呼びます。

---

## 値を返すメソッド

2つの整数を足して、その結果を返すメソッド `add` を作ってみましょう。

```java
int add(int a, int b) {
    return a + b;
}
```

前の節までと違う点が2つあります。

1. 先頭が `void` ではなく **`int`** … 「**`int` の値を返す**」という意味（これが戻り値の型）
2. 中に **`return`** がある … `return 値;` で、その値を呼び出し元へ返す

`return a + b;` は、「`a + b` を計算して、その結果を返す」という意味です。

呼び出すと、戻り値が、その場所に**置きかわります**。

```java
int sum = add(3, 5);
```

`add(3, 5)` が `8` を返すので、`sum` には `8` が入ります。

```text
jshell> int sum = add(3, 5);
sum ==> 8
```

---

## 戻り値の使い方

戻り値は、ふつうの値とまったく同じように使えます。変数に入れるだけでなく、

```java
IO.println(add(3, 5));        // 結果をそのまま表示 → 8
int total = add(3, 5) + 100;  // 計算の一部に使う → 108
```

このように、`add(3, 5)` と書いた場所が、戻り値 `8` に置きかわる、と考えるとわかりやすいでしょう。

戻り値の型は `int` だけではありません。返したいものに合わせて、どんな型でも指定できます。

```java
double average(int a, int b) {
    return (a + b) / 2.0;
}

boolean isEven(int n) {
    return n % 2 == 0;
}
```

```text
jshell> average(3, 4)
$1 ==> 3.5

jshell> isEven(4)
$2 ==> true
```

- `average` は `double` を返すので、小数の平均（`3.5`）が得られます
- `isEven`（偶数か？）は `boolean` を返すので、`true` / `false` が得られます。`if` の条件などにそのまま使えて便利です

---

## void は「値を返さない」

第2節で出てきた `void` は、**「戻り値がない」**という意味でした。
表示するだけのメソッドのように、返す値がないときは `void` を指定します。`void` のメソッドには、`return 値;` は書けません。

ただし、`void` のメソッドでも、**値をつけない `return;`** は書けます。これは「ここでメソッドを終える」という意味で、途中で抜けたいときに使います。

```java
void check(int n) {
    if (n < 0) {
        IO.println("マイナスです");
        return;            // ここで終了。下へは進まない
    }
    IO.println("0以上です");
}
```

```text
jshell> check(-3)
マイナスです

jshell> check(5)
0以上です
```

`n` がマイナスのときは `return;` で抜けるので、その先の「0以上です」は表示されません。

---

## return は複数書いてもよい

`return` は、メソッドの中に複数あっても構いません。
条件によって返す値を変えたいときに使います。最初に実行された `return` で、メソッドは終わります。

```java
String grade(int score) {
    if (score >= 80) {
        return "A";
    }
    if (score >= 60) {
        return "B";
    }
    return "C";
}
```

```text
jshell> grade(95)
$1 ==> "A"

jshell> grade(70)
$2 ==> "B"

jshell> grade(40)
$3 ==> "C"
```

`score` が 95 なら最初の `return "A";` で終わり、それ以降は実行されません。
このように、`return` には「**値を返す**」と「**メソッドをそこで終える**」の2つの働きがあります。

> **注意: 値を返すメソッドは、必ず値を返す**
>
> 戻り値の型が `void` でないメソッドは、**どんな場合でも必ず `return 値;` にたどり着く**必要があります。
> 返さない経路が1つでもあると、`return文が指定されていません`（missing return statement）というコンパイルエラーになります。詳しくは第8節で扱います。

---

## まとめ

- **戻り値**は、メソッドが呼び出し元へ返す値。先頭に**戻り値の型**を書き、`return 値;` で返す
- 呼び出した場所が、戻り値に**置きかわる**（変数に入れる・計算や表示に使う）
- 戻り値の型は `int`・`double`・`boolean`・`String` など自由
- **`void`** は値を返さない。値なしの `return;` で途中終了はできる
- `return` は複数書ける。最初に実行された `return` でメソッドは終わる
