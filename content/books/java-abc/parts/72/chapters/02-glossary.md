## 用語集 ― この章で学んだ言葉

| 用語 | 英語 | 意味 |
|---|---|---|
| Visitor | Visitor | 構造を変えずに操作を追加できるようにする振る舞いパターン |
| `accept` | accept | 構造側が訪問者を受け入れる入口メソッド |
| ダブルディスパッチ | Double Dispatch | 構造の具象型 × 訪問者の具象型で振る舞いを決める仕組み |
| Expression Problem | Expression Problem | 操作と構造のどちらの追加を楽にするかというトレードオフ |
| シールドクラス | Sealed Class | 派生型を限定し、網羅チェックを可能にする Java 17+ の機能 |

---

## この章で学んだこと

- **Visitor** は、**操作を増やす方向**の拡張を容易にする
- 古典実装は冗長。**シールドクラス + `switch` 式**が現代の有力な代用
- 拡張の方向（構造 vs 操作）を意識して選ぶ
