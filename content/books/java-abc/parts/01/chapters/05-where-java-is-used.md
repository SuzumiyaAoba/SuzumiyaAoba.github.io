## Java が使われている場所

Java は、世界でもっとも広く使われているプログラミング言語のひとつです。

しかし、ふだんの生活のなかで「これは Java で動いています」と表示されることはほとんどありません。
そのため、Java がどこで使われているのか、実感しにくいかもしれません。

実は Java は、私たちの目に見えないところで、社会のさまざまな仕組みを支えています。
この節では、Java が実際にどのような場所で活躍しているのかを見ていきましょう。

---

## 企業の業務システム・基幹システム

Java がもっとも力を発揮してきた分野のひとつが、**企業の業務システム**です。

たとえば、次のようなシステムです。

- 銀行の口座管理や振込の仕組み
- 保険会社の契約管理システム
- 航空券やホテルの予約システム
- 企業の在庫管理・販売管理システム
- 官公庁の各種手続きシステム

こうしたシステムは、会社や社会の根幹を支えるため、**基幹システム**（きかんシステム）とも呼ばれます。

基幹システムには、非常に高い信頼性が求められます。
たくさんの利用者が同時に使っても止まらず、長期間にわたって安定して動き続け、お金や個人情報といった重要なデータを正確に扱わなければなりません。

前の節で学んだ Java の特徴 ―― 堅牢性（安定して動きやすい）、後方互換性（古いプログラムが動き続けやすい）、豊富なライブラリ ―― は、まさにこうした用途にぴったりです。
そのため、長く使われる重要なシステムの多くで、Java が選ばれてきました。

## Web アプリケーションのサーバーサイド

私たちが日常的に使う Web サービスの裏側でも、Java は広く使われています。

ニュースサイトやネットショッピング、各種の Web サービスでは、

- 利用者からのリクエストを受け取る
- データベースから必要な情報を取り出す
- 計算や判断を行う
- 結果を画面に返す

といった処理が、利用者からは見えないコンピュータ（サーバー）の上で行われています。

この、**サーバー側で動くプログラム**のことを、**サーバーサイド**（Server-Side）と呼びます。

Java は、このサーバーサイドの開発で非常によく使われています。
とくに **Spring（スプリング）** をはじめとする強力な道具（フレームワーク）が充実しており、大規模で本格的な Web サービスを効率よく開発できます[^spring-framework]。

Spring を使った Web アプリケーションの作り方は、本書の第4部で学びます。

## Android アプリ

スマートフォンの **Android**（アンドロイド）向けアプリの開発でも、Java は長く中心的な役割を果たしてきました。

Android は、その仕組みの多くが Java をもとに作られており、長年にわたって Java が主要な開発言語とされてきました。

近年では、**Kotlin（コトリン）** という言語も広く使われるようになっています。
Kotlin は Java と深い関わりを持つ言語で、現在 Google は Android 開発で Kotlin を推奨しています[^android-kotlin-first]。

とはいえ、Kotlin は Java の考え方を土台にした言語であり、Android の仕組みそのものは今も Java と密接に結びついています。
そのため、Java を学ぶことは、Android アプリ開発を理解するうえでも、引き続き大きな助けになります。

## 大規模データ処理・分散システム

近年、世界中で生み出される膨大なデータ（ビッグデータ）を扱う技術が重要になっています。

こうした大規模なデータ処理の分野でも、Java は重要な役割を果たしています。

大量のデータを保存・検索・分析するための有名な基盤技術の多くは、**JVM 上で動く**ように作られています。
JVM（Java 仮想マシン）とは、Java のプログラムを実行する土台となる仕組みのことで、次の節でくわしく説明します。

これらの基盤技術が Java や JVM をもとに作られているのは、

- 大量の処理を安定してこなせる
- たくさんのコンピュータで分担して動かしやすい
- 性能と信頼性のバランスがよい

といった理由からです。

私たちが何気なく使っている検索やおすすめ機能の裏側でも、JVM 上で動く技術が活躍しているのです。

## クラウドと、その他の分野

現在では、多くのシステムが**クラウド**（インターネットを通じて提供されるコンピュータの仕組み）の上で動いています。

Java は、こうしたクラウド環境や、システムを小さな部品に分けて開発する手法（マイクロサービス）とも相性がよく、現在も活発に使われています。

このほかにも、Java は次のような幅広い分野で利用されています。

- 機器に組み込んで使う制御プログラム（組み込み・IoT）
- 科学技術計算や研究用のソフトウェア
- 開発を支援するさまざまな道具（ツール）

このように、Java は特定の分野に限られた言語ではありません。
小さな機器から、社会を支える巨大なシステムまで、非常に幅広い場面で使われている、汎用性の高い言語です。

## なぜ多くの場面で Java が選ばれるのか

これだけ幅広い分野で Java が使われている背景には、これまで学んできた Java の特徴があります。

- **安定して動く** ： 重要なシステムでも安心して使える堅牢性がある
- **長く使える** ： 後方互換性が高く、長期にわたって運用しやすい
- **どこでも動く** ： 環境を選ばずに動かせる
- **道具が豊富** ： 目的に合ったライブラリやフレームワークが見つかりやすい
- **使う人が多い** ： 利用者が多く、情報が豊富で、開発を担う人材も見つけやすい

とくに、長く使われ続けることが前提となるシステムでは、「安定して動き、長く保守できる」ことが何よりも重視されます。
Java は、この要求に長年こたえてきた実績があるため、今も多くの現場で選ばれ続けています。

Java を学ぶことは、こうした数多くの現場で通用する、実用的な力を身につけることにつながります。

## まとめ

この節では、Java が使われている場所を見てきました。

- **企業の業務システム・基幹システム** ： 銀行や保険など、止まると困る重要なシステムで広く使われている
- **Web アプリケーションのサーバーサイド** ： Spring などを使い、本格的な Web サービスの裏側を支えている
- **Android アプリ** ： 長く中心的な言語であり、Kotlin が普及した今も土台として重要
- **大規模データ処理・分散システム** ： ビッグデータを扱う基盤技術の多くが JVM 上で動いている
- **クラウドやその他の分野** ： 組み込みから科学計算まで、非常に幅広く使われている
- これらの背景には、安定性・互換性・汎用性・利用者の多さといった Java の強みがある

Java は、社会のさまざまな仕組みを、目に見えないところで支えている言語です。

次の節では、いよいよ Java の核心に踏み込みます。
「一度書けば、どこでも動く」を実現している、Java が動く仕組みを見ていきましょう。

[^spring-framework]: Spring, "Spring Framework," <https://spring.io/projects/spring-framework>。Spring は VMware（旧 Pivotal）が開発するエンタープライズ Java の代表的フレームワークで、Spring Boot を含むエコシステムが Java サーバサイド開発で広く採用されている。

[^android-kotlin-first]: Google Developers, "Android's commitment to Kotlin"（2019年 Google I/O での発表）, <https://developer.android.com/kotlin/first>。Google は2019年に「Android 開発は Kotlin ファースト」と公式表明し、以降の Android Jetpack API は Kotlin を優先的にサポートしている。
