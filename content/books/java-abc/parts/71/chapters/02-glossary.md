## 用語集 ― この章で学んだ言葉

| 用語 | 英語 | 意味 |
|---|---|---|
| Memento | Memento | 内部状態のスナップショットを保存・復元する振る舞いパターン |
| Originator | Originator | 状態を持つ本体。Memento を作り、戻す対象 |
| Caretaker | Caretaker | Memento を保持する管理役。中身は理解しない |
| スナップショット | Snapshot | ある時点の状態を丸ごと写し取ったもの |

---

## この章で学んだこと

- **Memento** は、状態を**カプセル化を保ったまま**保存するパターン
- 履歴・Undo・チェックポイントの実装に使える
- 現代の Java では、**レコード**でシンプルに表現できる
- Command（逆操作）との使い分けがある
