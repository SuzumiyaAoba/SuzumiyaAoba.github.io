## 用語集 ― この章で学んだ言葉

| 用語 | 英語 | 意味 |
|---|---|---|
| Composite | Composite | 葉と節を同じインターフェースで扱う構造パターン |
| 葉 | Leaf | 子を持たない要素（例: ファイル） |
| 節 | Composite / Node | 子要素を内包する複合要素（例: ディレクトリ） |
| 木構造 | Tree | 親子関係を持つ階層構造 |
| 再帰 | Recursion | 自分自身を呼び出して、構造をたどる手法 |

---

## この章で学んだこと

- **Composite** は木構造を一様に扱うパターン
- シールドクラス + レコード + パターンマッチングで、現代的に安全に書ける
- 「葉だけ／節だけの操作」をひとつのインターフェースに混ぜない
