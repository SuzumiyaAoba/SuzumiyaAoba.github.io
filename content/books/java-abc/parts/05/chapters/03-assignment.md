## 代入と複合代入・インクリメント

第4章で、`=` は「右の値を左の変数に入れる」**代入演算子**だと学びました。
この節では、その代入をもっと短く書ける、便利な記法を学びます。

---

## 自分自身を使って更新する

プログラムでは、「変数の値を、いまの値をもとに更新する」ことがよくあります。
たとえば、得点に 5 点を加える、という操作です。

```text
jshell> int score = 10;
score ==> 10
jshell> score = score + 5;
score ==> 15
```

`score = score + 5` は、「いまの score（10）に 5 を足した結果（15）を、score に入れ直す」という意味でした（第4章で学んだとおりです）。

---

## 複合代入演算子 ― += など

この「自分自身を使った更新」は、とてもよく使うので、短く書ける記法が用意されています。
それが**複合代入演算子**（Compound Assignment Operator）です。

`score = score + 5` は、`+=` を使って次のように書けます。

```java
score += 5;   // score = score + 5 と同じ
```

同じように、ひき算・かけ算などにも複合代入があります。

| 複合代入 | 同じ意味 |
|---|---|
| `x += 5` | `x = x + 5` |
| `x -= 5` | `x = x - 5` |
| `x *= 2` | `x = x * 2` |
| `x /= 2` | `x = x / 2` |
| `x %= 3` | `x = x % 3` |

どれも「自分自身に対して計算して、入れ直す」という操作を、短く書いたものです。

---

## インクリメントとデクリメント ― ++ と --

特に「1 を足す」「1 を引く」は、繰り返し（第7章）などで非常によく使います。
そのため、さらに短い専用の演算子があります。

- **`++`**（インクリメント）… 1 を足す。`i = i + 1` と同じ
- **`--`**（デクリメント）… 1 を引く。`i = i - 1` と同じ

```text
jshell> int i = 5;
i ==> 5
jshell> i = i + 1;
i ==> 6
```

この `i = i + 1` は、`i++` と短く書けます。

```java
i++;   // i = i + 1 と同じ（i が 1 増える）
i--;   // i = i - 1 と同じ（i が 1 減る）
```

---

## 前置（++i）と後置（i++）の違い

`++` には、変数の**後ろ**に付ける `i++`（後置）と、**前**に付ける `++i`（前置）の、2つの書き方があります。
「変数を1増やす」という結果は同じですが、**式の中で値として使ったとき**に返す値が違います。

- **後置 `i++`** … **増やす前**の値を返してから、1を足す
- **前置 `++i`** … 先に1を足してから、**増やしたあと**の値を返す

実際に確かめてみましょう。まず後置です。

```java
int a = 5;
int b = a++;   // 後置: b には「増やす前」の 5 が入り、その後 a が 6 になる
```

```text
jshell> a
a ==> 6
jshell> b
b ==> 5
```

次に前置です。

```java
int c = 5;
int d = ++c;   // 前置: 先に c が 6 になり、その 6 が d に入る
```

```text
jshell> c
c ==> 6
jshell> d
d ==> 6
```

`a++` では `b` が `5`（増やす前の値）、`++c` では `d` が `6`（増やした後の値）になりました。
`a` と `c` 自身は、どちらも最終的に `6` です。違うのは「式が返す値」だけ、という点に注目してください。

> **補足: ふだんは気にしなくてよい**
>
> この違いが問題になるのは、`b = a++` のように「増やしながら、その値を使う」ときだけです。
> `i++;` と単独の文で書くぶんには、前置でも後置でも同じです。
> 混乱を避けるため、まずは「`i++;` と単独で1増やす」使い方を基本にするとよいでしょう。

## まとめ

この節では、代入の便利な書き方を学びました。

- `x = x + 5` は、`x += 5` と短く書ける（**複合代入演算子**）。`-=`・`*=`・`/=`・`%=` も同様
- 「1 増やす」は `i++`、「1 減らす」は `i--`（**インクリメント**・**デクリメント**）
- `i++`（後置）と `++i`（前置）は、式が返す値が違う（単独の文で書くぶんには同じ）
- これらは、第7章の繰り返しで特によく使う
