## 式と演算子とは

計算や比較に使う記号を学ぶ前に、関連する3つの言葉を整理しておきましょう。
**演算子**・**被演算子**・**式**です。

---

## 演算子と被演算子

**演算子**（Operator）とは、計算や比較などの操作を表す記号です。
たとえば、足し算の `+`、大小を比べる `>` などが演算子です。

そして、演算子が操作する対象の値を、**被演算子**（ひえんざんし、Operand）と呼びます。

```text line-numbers=false
   7  +  3
   │  │  │
   │  │  └─ 被演算子
   │  └──── 演算子
   └─────── 被演算子
```

`7 + 3` では、`+` が演算子、その両側の `7` と `3` が被演算子です。

---

## 式 ― 値を生み出すまとまり

`7 + 3` のように、**計算されて1つの値になるまとまり**を、**式**（Expression）と呼びます。

式は、評価される（計算される）と、1つの値になります。jshell で確かめてみましょう。

```text line-numbers=false
jshell> 7 + 3
$1 ==> 10
```

`7 + 3` という式が評価され、`10` という値になりました。
（行頭の `$1` は、jshell が式の結果に付ける名前です。番号は気にしなくてかまいません。）

式は、変数を使っても作れます。

```text line-numbers=false
jshell> int price = 100;
price ==> 100
jshell> int total = price * 3;
total ==> 300
```

`price * 3` も式で、評価すると `300` になり、それが `total` に入りました。
このように、**式はどこかで必ず1つの値になる**、と覚えておきましょう。

---

## 演算子の種類

Java の演算子は、はたらきによって、いくつかの種類に分けられます。
この章では、次の演算子を順に学びます。

| 種類 | 例 | はたらき |
|---|---|---|
| 算術演算子 | `+` `-` `*` `/` `%` | 計算する |
| 代入演算子 | `=` `+=` `++` | 変数に値を入れる |
| 比較演算子 | `==` `<` `>` | 値を比べる（結果は `boolean`） |
| 論理演算子 | `&&` `\|\|` `!` | 真偽を組み合わせる |
| 条件演算子 | `? :` | 条件で値を選ぶ |

それぞれ、次の節から順に見ていきます。

## まとめ

この節では、演算子にまつわる基本の言葉を整理しました。

- **演算子**は、計算や比較を表す記号。**被演算子**は、演算子が操作する値
- **式**は、評価されて1つの値になるまとまり（`7 + 3` → `10`）
- 演算子には、算術・代入・比較・論理・条件などの種類がある
