## List の操作 ― 取得・変更・削除・検索

List には、要素を扱うための便利なメソッドが、たくさん用意されています。
この節では、よく使う操作 ―― **取得・変更・削除・検索** ―― を、順に見ていきます。

以下では、次のリストを例に使います。

```java
List<String> names = new ArrayList<>();
names.add("佐藤");
names.add("鈴木");
names.add("高橋");
```

---

## 要素を取得する ― get(位置)

指定した位置の要素を取り出すには、**`get(位置)`** を使います。
位置（インデックス）は、配列と同じく **`0` から**始まります。

```java
IO.println(names.get(0));   // 0番目
IO.println(names.get(1));   // 1番目
```

```text
佐藤
鈴木
```

配列では `names[0]` と書きましたが、List では `names.get(0)` と、**メソッド**で取り出します。
「0 番目が先頭」というルールは、配列とまったく同じです。

---

## 要素数を調べる ― size()

リストに入っている要素の数は、**`size()`** で分かります。

```java
IO.println(names.size());
```

```text
3
```

配列の `length` にあたるものです。
ただし配列と違い、`size()` は「**実際に入っている個数**」を返します。`add` や `remove` をするたびに、この値は自動で変わります。

---

## 要素を変更する ― set(位置, 値)

すでにある要素を、別の値に置きかえるには、**`set(位置, 値)`** を使います。

```java
names.set(1, "田中");   // 1番目を「田中」に変更
IO.println(names);
```

```text
[佐藤, 田中, 高橋]
```

`1` 番目の `鈴木` が、`田中` に置きかわりました。
`set` は「すでにある位置を、上書きする」操作です。新しく**増やす**のではない点に注意してください（増やすのは `add`）。

---

## 要素を削除する ― remove

要素を削除するには、**`remove(...)`** を使います。`remove` には、2つの使い方があります。

```java
names.remove("佐藤");   // 「佐藤」という値を削除
IO.println(names);
```

```text
[田中, 高橋]
```

`remove("佐藤")` は、「`佐藤` という**値**」を探して削除します。
削除されると、うしろの要素が**自動的に前へ詰められます**（配列のように、すきまが残ったりしません）。

位置を指定して削除することもできます。

```java
List<String> nums = new ArrayList<>();
nums.add("a"); nums.add("b"); nums.add("c");
nums.remove(0);          // 0番目を削除
IO.println(nums);
```

```text
[b, c]
```

> **注意: `List<Integer>` の remove は、ちょっと注意**
>
> `List<Integer>`（数値のリスト）で `remove(1)` と書くと、「**1 番目を削除**」なのか「**値 `1` を削除**」なのか、迷うところです。
> Java は、`int` を渡すと「**位置**（1 番目）の削除」と解釈します。「値 `1` を削除」したいなら、`remove(Integer.valueOf(1))` のように、はっきり `Integer` を渡します。
> 数値のリストでの `remove` は、この区別に注意しましょう（文字列のリストなら、この迷いはありません）。

---

## 要素を検索する ― contains / indexOf

「ある値が、リストに含まれているか」を調べるには、**`contains(値)`** を使います。結果は `boolean` です。

```java
List<String> names = new ArrayList<>();
names.add("佐藤"); names.add("鈴木"); names.add("高橋");

IO.println(names.contains("鈴木"));   // 含まれているか
IO.println(names.contains("田中"));   // 含まれていないか
```

```text
true
false
```

「値が、何番目にあるか」を知りたいときは、**`indexOf(値)`** を使います。

```java
IO.println(names.indexOf("高橋"));   // 何番目か
IO.println(names.indexOf("田中"));   // ないときは -1
```

```text
2
-1
```

`indexOf` は、見つかればその位置（0 始まり）を、見つからなければ **`-1`** を返します。
「-1 が返ったら、含まれていない」というのは、よくある約束ごとです。

---

## よく使う操作のまとめ

ここまでの操作を、表で整理しておきましょう。

| やりたいこと | メソッド | 例 |
|---|---|---|
| 末尾に追加 | `add(値)` | `names.add("田中")` |
| 取得 | `get(位置)` | `names.get(0)` |
| 個数 | `size()` | `names.size()` |
| 変更（上書き） | `set(位置, 値)` | `names.set(1, "田中")` |
| 削除 | `remove(値)` / `remove(位置)` | `names.remove("佐藤")` |
| 含むか | `contains(値)` | `names.contains("鈴木")` |
| 位置を調べる | `indexOf(値)` | `names.indexOf("高橋")` |
| 空かどうか | `isEmpty()` | `names.isEmpty()` |
| 全部消す | `clear()` | `names.clear()` |

これだけ覚えておけば、List の基本操作は、ほぼカバーできます。
どれも「`リスト.メソッド(...)`」という、第11章で学んだメソッド呼び出しの形です。

---

## まとめ

- **`get(位置)`** で取得、**`set(位置, 値)`** で変更（位置は `0` 始まり）
- **`size()`** で要素数（実際の個数。`add`/`remove` で自動的に変わる）
- **`remove(値)`** で値を削除、**`remove(位置)`** で位置を削除（削除後は自動で詰められる）
- **`contains(値)`** で含むか判定、**`indexOf(値)`** で位置を取得（なければ `-1`）
- `List<Integer>` の `remove` は「位置」と解釈される点に注意
