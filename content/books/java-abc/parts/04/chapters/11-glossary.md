## 用語集 ― この章で学んだ言葉

この第4章でも、たくさんの新しい言葉が登場しました。

ここでは、それらの用語を**節ごとに**まとめておきます。
「あの言葉は何だったかな」と思ったときに、辞書のように使ってください。

---

## 変数（第1〜2節）

| 用語 | 英語・読み | 意味 |
|---|---|---|
| 変数 | Variable | 値に名前をつけて保存する仕組み。「名前をつけた箱」 |
| 宣言 | Declaration | 変数（箱）を用意すること（例: `int age;`） |
| 初期化 | Initialization | 変数に最初の値を入れること（例: `int age = 20;`） |
| 代入 | Assignment | 変数に値を入れること。`=` を使う |
| 代入演算子 | `=` | 右の値を左の変数に入れる記号。「等しい」ではない |

---

## データ型（第3〜6節）

| 用語 | 英語・読み | 意味 |
|---|---|---|
| データ型（型） | Data Type | 変数という箱の「種類」 |
| 基本データ型 | プリミティブ型、Primitive Type | `int`・`double`・`boolean`・`char` など、値そのものを直接入れる型 |
| 参照型 | Reference Type | `String` など、基本データ型以外の型。くわしくは第2部で学ぶ |
| 静的型付け | Static Typing | 変数の型をあらかじめ決め、合わない値を入れさせないしくみ |
| `int` | イント | 整数を表す基本データ型（約 -21 億〜21 億） |
| `long` | ロング | 大きな整数を表す型。リテラルの末尾に `L` を付ける |
| `double` | ダブル | 小数を表す基本データ型 |
| `boolean` | ブーリアン | 真偽（`true` / `false`）を表す型 |
| `char` | チャー | 1文字を表す型。シングルクォート `'` で囲む |
| `String` | ストリング | 文字のまとまりを扱う型（参照型）。ダブルクォート `"` で囲む |
| 連結 | ― | `+` で文字列どうしや、文字列と値をつなげること |
| メソッド | Method | 値に対して行える操作（例: `length()`）。第9章で学ぶ |
| リテラル | Literal | ソースコードに直接書いた値（`100`、`"Hello"` など） |

---

## 型変換（第7節）

| 用語 | 英語・読み | 意味 |
|---|---|---|
| 型変換 | Type Conversion | ある型の値を、別の型に変えること |
| 自動変換 | ― | 狭い型から広い型へ（`int`→`double` など）、自動で行われる変換 |
| キャスト | Cast | 広い型から狭い型へ、`(int)` のように明示的に行う変換（小数は切り捨て） |
| 桁あふれ | オーバーフロー、Overflow | 型の範囲を超えて値があふれること |

---

## var・final・命名（第8〜9節）

| 用語 | 英語・読み | 意味 |
|---|---|---|
| 型推論 | Type Inference | 右の値から型を判断するしくみ。`var` で使う |
| `var` | バー | 型を書くかわりに使う。Java が型を推論してくれる |
| `final` | ファイナル | 変数を、書き換えられない**定数**にする |
| 定数 | Constant | 書き換えない値。`final` で作り、名前は大文字（`MAX_SCORE`） |
| 識別子 | Identifier | 変数などにつける名前 |
| 予約語 | Reserved Word | Java があらかじめ使っている言葉（`int`・`class` など）。名前には使えない |
| キャメルケース | Camel Case | 変数名の慣習。小文字で始め、2語目以降の先頭を大文字にする（`userAge`） |

---

## おわりに

この第4章では、プログラムの「中身」を作る第一歩として、**変数とデータ型**を学びました。

- 変数は、値に名前をつけて保存する「箱」であること
- 値には種類（データ型）があり、`int`・`double`・`boolean`・`char`・`String` などを使い分けること
- 型変換やキャスト、整数の割り算の落とし穴
- `var`・`final` といった、より実践的な書き方と、わかりやすい名前のつけ方

そして、この章からは **jshell** を使い、実際に値を入れて動かしながら学んできました。
手を動かして「`age ==> 20`」という返事を確かめることで、変数のイメージが、より確かなものになったのではないでしょうか。

次の第5章「**演算子と式**」では、ここで覚えた変数や値を使って、**計算したり、比べたり**する方法を学びます。
変数に入れた値を、いよいよ「活用」していく段階です。
