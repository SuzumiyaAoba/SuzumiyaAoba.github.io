## データ型とは ― なぜ型が必要か

前の2節では、`int`（整数）の箱を例に、変数の使い方を学びました。
この `int` のような、**箱の種類**を表すものを、**データ型**（Data Type）、または単に**型**（Type）と呼びます。

この節では、「そもそも、なぜ箱に種類があるのか」を考えていきます。

---

## 値には「種類」がある

私たちが扱う値には、いろいろな種類があります。

- 年齢や個数のような **整数**（`20`、`5`、…）
- 身長や金額のような **小数**（`1.72`、`100.5`、…）
- はい／いいえのような **真偽**（`true`、`false`）
- 名前やメッセージのような **文字・文字列**（`"太郎"`、`"こんにちは"`）

これらは、性質がまったく違います。
たとえば「整数どうしを足し算する」ことには意味がありますが、「真偽（はい／いいえ）を足し算する」のは意味がありません。

そこで Java では、**値の種類ごとに、専用の箱（型）を用意**しています。
箱に「これは整数を入れる箱」「これは文字を入れる箱」と種類を決めておくことで、それぞれの値を正しく扱えるようにしているのです。

---

## 型があると、まちがいを防げる

型の大切な役割の一つが、**まちがった値が入るのを防ぐ**ことです。

整数を入れるための `int` の箱に、文字を入れようとすると、どうなるでしょうか。jshell で試してみましょう。

```text
jshell> int bad = "abc";
|  エラー:
|  不適合な型: java.lang.Stringをintに変換できません:
|  int bad = "abc";
|            ^---^
```

`不適合な型`（incompatible types）というエラーが出ました。
「整数の箱（`int`）に、文字（`String`）は入れられません」と、Java が教えてくれています。

一見、エラーは「失敗」のようでこわく感じるかもしれません。
ですが、これはとても**ありがたい**しくみです。
もし型のチェックがなければ、まちがった値が入ったことに気づかないまま、プログラムがおかしな動きをしてしまいます。
型があるおかげで、Java は**プログラムを動かす前の段階で**、こうしたまちがいを見つけて教えてくれるのです。

> **補足: Java は「型にきびしい」言語**
>
> Java は、変数を宣言するときに必ず型を決め、その型に合わない値を入れさせない言語です。
> このような言語を、**静的型付け**（せいてきかたづけ、Static Typing）の言語と呼びます。
> 最初は「いちいち型を書くのは面倒」と感じるかもしれませんが、そのぶん、まちがいを早く見つけられるという大きな利点があります。

---

## 型は、大きく2種類に分かれる

Java の型は、大きく2つのグループに分かれます。

| グループ | よび名 | 例 | 特徴 |
|---|---|---|---|
| 基本データ型 | プリミティブ型（Primitive Type） | `int`・`double`・`boolean`・`char` | 値そのものを、直接箱に入れる |
| それ以外の型 | 参照型（Reference Type） | `String` など | くわしくは第2部で学ぶ |

![Java の型の分類。データ型は、基本データ型（int・double・boolean・char など）と、参照型（String など）に分かれる。](./images/type-categories.svg)

この章では、まず使う機会の多い**基本データ型**を、次の第4節でくわしく見ていきます。
そして、文字列を扱う `String` は参照型の仲間ですが、とてもよく使うので、第5節で特別に取り上げます。

> **補足: いまは「2種類ある」とだけ知っておけば十分**
>
> 「基本データ型」と「参照型」の本当の違い（値そのものを持つか、値のある場所を指し示すか）は、変数やメモリのしくみを知るとはっきりします。
> これは第2部のクラスの章でくわしく扱いますので、いまは「型には大きく2つの種類があるんだな」とだけ知っておけば大丈夫です。

---

## まとめ

この節では、データ型とは何か、なぜ必要かを学びました。

- **データ型（型）**とは、変数という箱の「種類」のこと
- 値には整数・小数・真偽・文字などの種類があり、Java は種類ごとに専用の型を用意している
- 型があると、まちがった値が入るのを**プログラムを動かす前に**防げる（例: `int` に文字を入れるとエラー）
- Java は型にきびしい**静的型付け**の言語
- 型は大きく**基本データ型（プリミティブ型）**と**参照型**に分かれる

次の節では、もっともよく使う**基本データ型**（整数・小数・真偽・文字）を、一つずつ見ていきます。
