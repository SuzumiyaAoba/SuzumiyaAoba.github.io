## while 文 ― 条件が成り立つ間

**while 文**（ホワイルぶん）は、**条件が成り立つ（`true` の）あいだ、処理を繰り返す**もっとも基本的な繰り返しです。
`while` は英語で「〜のあいだ」という意味です。

---

## while の基本の形

```java
while (条件) {
    // 条件が true のあいだ、繰り返す処理
}
```

`if` とよく似ていますが、`if` が「一度だけ」だったのに対し、`while` は**条件が `true` であるかぎり、何度も**繰り返します。
1回ごとに条件を調べ、`false` になったら繰り返しを終えます。

実際に、0 から 2 までを表示してみましょう。

```java
int w = 0;
while (w < 3) {
    IO.println(w);
    w++;
}
```

```text line-numbers=false
0
1
2
```

流れを追ってみましょう。

1. `w` は 0。`w < 3` は `true` → `0` を表示し、`w++` で `w` は 1
2. `w` は 1。`w < 3` は `true` → `1` を表示し、`w` は 2
3. `w` は 2。`w < 3` は `true` → `2` を表示し、`w` は 3
4. `w` は 3。`w < 3` は `false` → 繰り返し終了

ここで、`w` を「開始（0）」「続ける条件（`w < 3`）」「更新（`w++`）」という、前節の3要素が、そろっているのがわかります。

---

## while らしい例 ― 終わりの回数が決まっていない繰り返し

前節で見たように、回数が決まった繰り返しは `for` が得意です。
一方 `while` は、「**何回繰り返すか、前もってわからない**」繰り返しで力を発揮します。

たとえば、「1 を 2 倍し続けて、はじめて 100 以上になるのはいくつか」を求めてみましょう。

```java
int n = 1;
while (n < 100) {
    n = n * 2;
}
```

```text line-numbers=false
jshell> n
n ==> 128
```

`n` は 1 → 2 → 4 → 8 → … と倍々に増え、`64` の次の `128` で、はじめて 100 以上になり、ループが終わります。
「何回でそうなるか」を自分で数えなくても、**続ける条件さえ書けば、コンピュータが必要なだけ繰り返してくれる** ―― これが while の便利なところです。

---

## 更新を忘れると、無限ループ

while 文で特に気をつけたいのが、**更新（`w++`）を忘れない**ことです。

もし `w++` を書き忘れると、`w` はずっと 0 のままで、`w < 3` が永遠に `true` になり、**無限ループ**になってしまいます。

```java
int w = 0;
while (w < 3) {
    IO.println(w);
    // w++; を書き忘れると、w は 0 のまま → 永遠に終わらない！
}
```

「繰り返しの中で、条件に関わる値を必ず更新する」――これを忘れないようにしましょう。

> **補足: 止まらなくなったら**
>
> もし無限ループでプログラムが止まらなくなったら、ターミナルで **Ctrl + C**（コントロールキーを押しながら C）を押すと、強制的に止められます。
> jshell の場合も、同じく Ctrl + C で実行中の処理を中断できます。

---

## まとめ

- **while 文**は、条件が `true` のあいだ、処理を繰り返す
- 1回ごとに条件を調べ、`false` になったら終わる
- 繰り返しの中で、**条件に関わる値を必ず更新する**（忘れると無限ループ）
- 止まらなくなったら、Ctrl + C で中断できる
