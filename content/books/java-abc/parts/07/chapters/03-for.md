## for 文 ― 回数が決まった繰り返し

**for 文**（フォーぶん）は、繰り返しの3要素（開始・条件・更新）を**1行にまとめて書ける**繰り返しです。
「N 回繰り返す」「1 から 10 まで」のように、**回数や範囲が決まっている**ときに、特に便利です。

---

## for の基本の形

```java
for (初期化; 条件; 更新) {
    // 繰り返す処理
}
```

かっこの中に、セミコロン `;` で区切って3つを書きます[^jls-for-statement]。

- **初期化** … 最初に1回だけ行う（例: `int i = 1`）
- **条件** … 繰り返しを続ける条件（例: `i <= 3`）
- **更新** … 1回ごとに行う（例: `i++`）

1 から 3 までを表示してみましょう。

```java
for (int i = 1; i <= 3; i++) {
    IO.println(i);
}
```

```text
1
2
3
```

`while` で書いた前節の3要素が、`for` では1行にきれいに収まっているのがわかります。
回数が決まった繰り返しでは、`while` よりも `for` のほうが、要素がそろっていて読みやすくなります。

for 文の実行順序を、図でも確認しておきましょう。

![for 文の実行順序。初期化は最初の1回だけ。そのあと「条件の判定 → 本体の処理 → 更新」を、条件が false になるまで繰り返す。](./images/for-flow.svg)

ポイントは、**初期化（`int i = 1`）は最初の1回だけ**実行され、そのあとは「**条件 → 本体 → 更新**」がぐるぐる繰り返される点です。更新（`i++`）は、本体の**あと**に実行されることにも注目してください。

## 合計を求める

for 文のよくある使い方が、**合計を求める**ことです。1 から 10 までの合計を計算してみましょう。

```java
int sum = 0;
for (int i = 1; i <= 10; i++) {
    sum += i;
}
```

ループのあと、`sum` を確認すると、

```text
jshell> sum
sum ==> 55
```

`sum` に、1 + 2 + … + 10 = 55 が入りました。
`sum` のように、繰り返しのなかで値をためていく変数を、しばしば**集計用の変数**として使います。

## 積み上げのもう一つの例 ― 階乗

合計（足し算の積み上げ）と同じ考え方で、**かけ算**を積み上げると、**階乗**（かいじょう）が計算できます。
階乗とは「1 × 2 × 3 × … × n」のことで、`5` の階乗（`5!` と書きます）は `1 × 2 × 3 × 4 × 5 = 120` です。

```java
int factorial = 1;
for (int i = 1; i <= 5; i++) {
    factorial *= i;
}
```

```text
jshell> factorial
factorial ==> 120
```

集計用の変数 `factorial` の初期値が、合計のときの `0` ではなく **`1`** になっている点に注目してください。
かけ算では、`0` から始めると何をかけても `0` になってしまうため、かけ算で「かけても変わらない値」である `1` から始めます。

## ループ変数 i について

for 文で使う `i` のような変数を、**ループ変数**（ループカウンター）と呼びます。
慣習として `i`、次に `j`、`k` …という名前がよく使われます（index の頭文字に由来します）。

`int i = 1` のように for のかっこの中で宣言したループ変数は、**その for 文の中だけ**で使えます。
ループが終わると `i` は使えなくなるので、ループの外で値を使いたいときは、`sum` のように別の変数を用意します。

> **補足: for と while の使い分け**
>
> `for` と `while` は、どちらも繰り返しですが、向き不向きがあります。
> **回数や範囲が決まっている**（「10 回」「1 から n まで」）なら `for`、**いつ終わるか決まっていない**（「正しい値が入力されるまで」など）なら `while` が読みやすくなります。
> どちらでも書ける場面も多いので、迷ったら「回数が決まっているか?」を基準に選ぶとよいでしょう。

## まとめ

- **for 文**は、繰り返しの3要素（初期化・条件・更新）を1行にまとめて書ける
- `for (int i = 1; i <= 3; i++)` のように、回数や範囲が決まった繰り返しに便利
- 実行順序は、**初期化（1回だけ）→ 条件 → 本体 → 更新 → 条件…** の繰り返し
- 合計や階乗を求めるなど、**集計用の変数**と組み合わせてよく使う
- for のかっこ内で宣言した**ループ変数**は、その for の中だけで使える

次の節では、先に1回実行してから条件を調べる **do-while 文**を学びます。

[^jls-for-statement]: *The Java® Language Specification, Java SE 25 Edition*, §14.14.1 "The basic for Statement," [https://docs.oracle.com/javase/specs/jls/se25/html/jls-14.html#jls-14.14.1](<https://docs.oracle.com/javase/specs/jls/se25/html/jls-14.html#jls-14.14.1>)。`for` 文は `ForInit`／`Expression`（条件）／`ForUpdate` の3要素で構成され、条件式が `false` になるまで本体を繰り返す。`ForInit` で宣言された変数のスコープは for 文の本体内に限定される。
