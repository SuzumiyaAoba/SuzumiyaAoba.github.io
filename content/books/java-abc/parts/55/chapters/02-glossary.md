## 用語集 ― この章で学んだ言葉

| 用語 | 英語 | 意味 |
|---|---|---|
| Prototype | Prototype | 原型となるオブジェクトをコピーして新しいインスタンスを作る生成パターン |
| 浅いコピー | Shallow Copy | 参照型のフィールドを共有する形でのコピー |
| 深いコピー | Deep Copy | 参照型のフィールドも含めて再帰的に複製するコピー |
| `Cloneable` | Cloneable | Java 標準の複製マーカーインターフェース（現代では非推奨） |
| コピーコンストラクタ | Copy Constructor | 同じ型のインスタンスを引数に取り、コピーを返すコンストラクタ |
| `with` メソッド | With Method | レコードなどで、一部だけ値を変えた新しいインスタンスを返すメソッドの慣習 |

---

## この章で学んだこと

- **Prototype** は、原型から複製してインスタンスを得るパターン
- 深さ（浅い/深い）の意識が、実装を成功させる鍵
- 現代の Java では `Cloneable` を避け、**コピーコンストラクタ**や **`record`+`with`** が定石
- これで生成パターン（第51〜55章）は完了。次は構造パターンへ
