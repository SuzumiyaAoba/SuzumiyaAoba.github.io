## 用語集 ― この章で学んだ言葉

この第10章で登場した言葉を、まとめておきます。

---

## 文字列の基本（第1〜2節）

| 用語 | 英語・読み | 意味 |
|---|---|---|
| String | ストリング | 文字の並びを表す型（ダブルクォート `"..."`） |
| char | キャラ／チャー | 1文字を表す型（シングルクォート `'...'`） |
| 不変 | Immutable（イミュータブル） | 一度作ると中身を変えられない性質。`String` はこれ |
| 文字列プール | String Pool | 同じ内容のリテラルを使い回す Java のしくみ |

---

## おもなメソッド早見表（第2〜6節）

| メソッド | 働き | 例 → 結果 |
|---|---|---|
| `length()` | 文字数 | `"abc".length()` → `3` |
| `charAt(i)` | i 番目の文字 | `"abc".charAt(0)` → `'a'` |
| `contains(t)` | t を含むか | `"abc".contains("b")` → `true` |
| `indexOf(t)` | t の位置（なければ `-1`） | `"abc".indexOf("c")` → `2` |
| `startsWith(t)` / `endsWith(t)` | t で始まる／終わるか | `"abc".endsWith("c")` → `true` |
| `isEmpty()` / `isBlank()` | 空か／空白だけか | `"".isEmpty()` → `true` |
| `substring(s, e)` | s から e の手前まで切り出す | `"abcd".substring(1, 3)` → `"bc"` |
| `replace(a, b)` | a を b にすべて置換 | `"aaa".replace("a", "b")` → `"bbb"` |
| `toUpperCase()` / `toLowerCase()` | 大文字／小文字に | `"Hi".toUpperCase()` → `"HI"` |
| `strip()` | 前後の空白を除く | `"  hi  ".strip()` → `"hi"` |
| `repeat(n)` | n 回繰り返す | `"ab".repeat(2)` → `"abab"` |
| `split(d)` | d で区切って配列に | `"a,b".split(",")` → `["a", "b"]` |
| `String.join(d, ...)` | d をはさんで結合 | `String.join("-", "a", "b")` → `"a-b"` |
| `formatted(...)` | 値を埋めて整形 | `"%d点".formatted(80)` → `"80点"` |
| `equals(t)` | 中身が同じか | `"abc".equals("abc")` → `true` |
| `equalsIgnoreCase(t)` | 大小無視で同じか | `"ABC".equalsIgnoreCase("abc")` → `true` |
| `compareTo(t)` | 辞書順の比較（負/0/正） | `"a".compareTo("b")` → 負の値 |

> すべての**加工メソッド**（`substring`・`replace`・`toUpperCase`・`strip` など）は、元を変えず**新しい文字列を返す**。結果は変数で受け取る。

## おわりに ― 第1部の修了

この第10章では、文字列を自在にあやつる**メソッド**を学びました。

- `String` は**不変**。加工メソッドは新しい文字列を返す
- `length()`・`charAt()`・`substring()`・`split()` など、豊富なメソッドで調べ・加工・整形できる
- 文字列の比較は、**必ず `equals`**（`==` は使わない）

そして、これで **第1部「プログラミングと Java の基礎」が修了**です。
ここまでで、次のことができるようになりました。

- 変数とデータ型で、値を扱う（第4章）
- 演算子・制御構文・繰り返しで、処理の流れを組み立てる（第5〜7章）
- 配列で、たくさんのデータをまとめる（第8章）
- メソッドで、処理を部品にする（第9章）
- 文字列を、思いどおりに操作する（第10章）

これらは、どんなプログラムにも共通する**基礎の土台**です。

次の**第2部「オブジェクト指向と Java の型システム」**では、いよいよ Java の中心となる考え方 ―― **オブジェクト指向**へと進みます。
第11章「**クラスとオブジェクト**」では、データ（変数）と処理（メソッド）を1つにまとめた「クラス」を、自分で設計できるようになります。
ここまで積み上げてきた基礎が、その土台になります。
