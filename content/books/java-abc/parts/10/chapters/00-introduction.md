## はじめに ― この章で学ぶこと

第9章では、処理を部品にする**メソッド**を学びました。
この章では、メソッドを使って、**文字列（String）を自在にあやつる**方法を学びます。

文字列は、第4章で「文字のまとまりを扱う型」として登場しました。
私たちはこれまで、`"こんにちは"` のように文字列を作って表示したり、`+` でつなげたりしてきました。
しかし、実際のプログラムでは、文字列に対してもっといろいろなことをします。

- 「この文字列に "@" は含まれているか？」を調べる
- 「最初の3文字だけ」を取り出す
- 「全部を大文字に」変換する
- 「カンマで区切って」バラバラにする
- 2つの文字列が「同じ内容か」を比べる

こうした操作は、すべて `String` が持つ**メソッド**で行えます。
第9章で「メソッドを呼び出す」しくみを学んだので、今度はそれを `String` で存分に使っていきましょう。

特に、文字列の**比較**（`==` と `equals` の違い）は、初心者が最もつまずきやすく、かつ実務でも重要なポイントです。この章でしっかり身につけます。

## この章で学ぶこと

第10章は、次の7つのセクションで構成されています。

| 節 | タイトル | 内容 |
|---|---|---|
| 1 | 文字列のおさらいと不変性 | `String` と `char`、そして「変えられない」性質 |
| 2 | 長さと1文字の取り出し | `length()` と `charAt()` |
| 3 | 文字列を調べる | 含むか・どこにあるか・始まり/終わり |
| 4 | 文字列を加工する | 切り出し・置き換え・大文字小文字 |
| 5 | 連結・分割・整形 | つなぐ・区切る・体裁を整える |
| 6 | 文字列の比較 | `==` と `equals` の違い（最重要） |
| 7 | よくあるつまずき | 文字列でつまずきやすい点と、その対処 |

この章は、**第1部「プログラミングと Java の基礎」の最終章**です。
ここまでで学んだ変数・演算子・制御構文・繰り返し・配列・メソッドを、文字列という身近な題材で総ざらいしながら進めます。

引き続き、サンプルは jshell に貼っても、ファイルに書いても試せます。
それでは、最初のセクション「文字列のおさらいと不変性」から始めます。
