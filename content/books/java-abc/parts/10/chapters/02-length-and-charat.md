## 長さと1文字の取り出し

文字列の基本操作として、「**何文字あるか**」を調べ、「**何番目の文字か**」を取り出す方法を学びます。

---

## 文字数を調べる ― length()

文字列の長さ（文字数）は、**`length()`** で得られます。

```java
String s = "Hello";
```

```text
jshell> s.length()
$2 ==> 5
```

`"Hello"` は5文字なので、`5` が返ります。

> **配列の `length` との違い（第8章の回収）**
>
> 第8章で予告したとおり、配列の長さは `length`（かっこなし）、文字列の長さは `length()`（**かっこあり**）です。
> 文字列の `length()` は**メソッド**（第9章で学んだ「呼び出すしくみ」）なので、`()` が付きます。
> 「配列はかっこなし、文字列はかっこあり」と、あらためて覚えておきましょう。

なお、空文字列 `""` の長さは `0` です。`" "`（スペース1つ）は、見えませんが1文字なので、長さは `1` です。

---

## 1文字を取り出す ― charAt()

指定した位置の文字を取り出すには、**`charAt(インデックス)`** を使います。
`charAt` は「char at（〜の位置の文字）」という意味です。インデックスは、配列と同じく **`0` から始まります**。

```java
String s = "Hello";
```

```text
jshell> s.charAt(0)
$2 ==> 'H'

jshell> s.charAt(1)
$3 ==> 'e'

jshell> s.charAt(4)
$4 ==> 'o'
```

返ってくるのは `char`（1文字）なので、結果が `'H'` のようにシングルクォートで表示されます。
前節で学んだとおり、文字列は `char` の並びでした。`charAt` は、その並びから1文字を取り出すメソッドです。

最後の文字を取り出したいときは、`charAt(s.length() - 1)` とします。
長さが5なら、最後のインデックスは `4`（＝長さ − 1）だからです。配列と同じ考え方ですね。

```text
jshell> s.charAt(s.length() - 1)
$5 ==> 'o'
```

---

## 1文字ずつ処理する

`length()` と `charAt()` を、第7章の **for 文**と組み合わせると、文字列を**1文字ずつ**処理できます。

```java
String s = "Java";
for (int i = 0; i < s.length(); i++) {
    IO.println(i + ": " + s.charAt(i));
}
```

```text
0: J
1: a
2: v
3: a
```

`i` を `0` から `s.length() - 1`（＝3）まで動かし、`s.charAt(i)` でその位置の文字を取り出しています。
配列を for で回したのと、まったく同じ形です。文字列も「文字が並んだもの」として、繰り返しで処理できるわけです。

---

## まとめ

- 文字数は **`length()`**（文字列は**かっこあり**。配列の `length` と区別）
- 指定位置の文字は **`charAt(インデックス)`**。インデックスは `0` 始まりで、`char` が返る
- 最後の文字は `charAt(length() - 1)`
- `length()` と `charAt()` を for と組み合わせると、**1文字ずつ**処理できる
