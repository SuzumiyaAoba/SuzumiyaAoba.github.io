## Path と Files ― ファイルの場所と操作

ファイル入出力の主役は、2つのクラスです。
**`Path`**（パス）が「**ファイルの場所**」を表し、**`Files`**（ファイルズ）が「**ファイルの操作**」を担当します。
この節では、この2つの役割を学びます。

---

## Path ― ファイルの「場所」

**`Path`** は、ファイルやフォルダの「**場所（位置）**」を表します。
**`Path.of("パス")`** で作ります。

```java
Path path = Path.of("memo.txt");
IO.println(path);
```

```text
memo.txt
```

ここで大切なのは、`Path` は「**場所を表しているだけ**」で、まだファイルを読んだり書いたりはしていない、ということです。
「memo.txt という場所を指す、住所メモ」のようなもの、と考えてください。
実際にそのファイルが存在するかどうかも、この時点では問いません。

パスには、フォルダを含めることもできます。

```java
Path path = Path.of("data/users.csv");   // data フォルダの中の users.csv
```

`/` で区切って、フォルダの階層を表します。

---

## 相対パスと絶対パス

パスには、2種類の書き方があります。

| 種類 | 意味 | 例 |
|---|---|---|
| **相対パス** | 実行している場所からの、相対的な位置 | `memo.txt`、`data/users.csv` |
| **絶対パス** | ルートからの、完全な位置 | `/Users/taro/memo.txt` |

- **相対パス** … 「いまいる場所（プログラムを実行したフォルダ）」を基準にした、近道の住所
- **絶対パス** … 「いちばん上（ルート）」からの、完全な住所

ふだんは、**相対パス**を使うことがほとんどです。
「プログラムを実行したフォルダにある `memo.txt`」のように、手軽に指定できます。

---

## Path から、情報を取り出す

`Path` からは、「ファイル名」や「親フォルダ」を取り出せます。

```java
Path path = Path.of("data/users.csv");

IO.println(path.getFileName());   // ファイル名
IO.println(path.getParent());     // 親フォルダ
```

```text
users.csv
data
```

- `getFileName()` … ファイル名の部分（`users.csv`）
- `getParent()` … 親フォルダの部分（`data`）

パスを部品に分けたいときに、便利です。

---

## Files ― ファイルの「操作」

**`Files`** は、ファイルに対する**操作**（読む・書く・存在確認・削除など）を提供する道具です。
`Files` のメソッドに `Path` を渡して、操作します。

```text
Path（場所） + Files（操作） = ファイル入出力

  Files.writeString(path, "...")  … path の場所に書き込む
  Files.readString(path)          … path の場所から読み込む
  Files.exists(path)              … path の場所にファイルがあるか
```

`Files` のメソッドは、すべて **`static`**（第11章）です。
そのため、`new Files()` のようにオブジェクトを作る必要はなく、`Files.メソッド名(...)` と、クラス名から直接呼びます（第8章の `Arrays`、第11章の `Math` と同じです）。

具体的な読み書きは、次の節から学びます。
ここでは、「**場所は `Path`、操作は `Files`**」という役割分担を、つかんでおいてください。

---

## まとめ

- **`Path`** … ファイルやフォルダの「**場所**」を表す（`Path.of("memo.txt")`）
- `Path` は場所を指すだけ。実際の読み書きはまだしていない
- **相対パス**（実行場所が基準）と**絶対パス**（ルートが基準）がある。ふだんは相対パス
- `getFileName()`・`getParent()` で、パスの部品を取り出せる
- **`Files`** … ファイルの「**操作**」を担う道具（`static` メソッド。`Path` を渡して使う）
- 役割分担は「**場所は `Path`、操作は `Files`**」
