## 第1章: プログラミングとは・Java とは {#chapter1}

はじめに ― この章で学ぶこと

1. プログラムとは何か
2. コンピュータの基本的な仕組み（CPU・メモリ・ストレージ）
3. Java の誕生と歩み
4. Java の特徴（プラットフォーム非依存・オブジェクト指向・堅牢性）
5. Java が使われている場所（Web・Android・企業システム）
6. Java の動く仕組み ― コンパイルと JVM・バイトコード
7. JDK / JRE / JVM の違い
8. Java 25 について ― LTS とリリースサイクル

用語集 ― この章で学んだ言葉

## 第2章: 環境構築 {#chapter2}

はじめに ― この章で学ぶこと

1. 開発環境の全体像 ― 何を準備するのか
2. JDK を選ぶ ― ディストリビューションの違い
3. JDK をインストールする（Windows / macOS / Linux）
4. インストールを確認する ― java --version と PATH
5. IDE を選ぶ・準備する（IntelliJ IDEA を中心に）
6. IDE で最初のプログラムを実行する
7. コマンドラインで実行する（javac / java / jshell）
8. うまくいかないとき ― よくあるトラブルと対処

用語集 ― この章で学んだ言葉

## 第3章: はじめてのプログラム {#chapter3}

はじめに ― この章で学ぶこと

1. プログラムの全体像をつかむ ― Hello, World! を部品に分ける
2. クラス ― プログラムをまとめる単位
3. main メソッド ― プログラムが始まる場所
4. 画面に文字を表示する ― System.out.println
5. 文・ブロック・セミコロン ― 文法の骨組み
6. コメント ― コードにメモを残す
7. エラーと向き合う ― はじめてのコンパイルエラー
8. Java 25 のシンプルな書き方 ― コンパクトソースファイルとインスタンス main
